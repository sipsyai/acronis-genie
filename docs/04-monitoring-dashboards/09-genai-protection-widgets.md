# GenAI protection widgets

GenAI protection widgets provide visibility into how endpoints access generative AI services. Use them to identify the most popular AI solutions, the devices with the highest activity, and recent usage details.

Widgets related to GenAI Protection are available in both Management Portal and the Cyber Protect console. The widgets that you can add and view depend on your role and on the currently selected level -- partner or customer.

## Top AI solutions (visits)

This widget shows the statistics of Generative AI usage by solution. It displays a donut chart with the total visits and a breakdown by AI solution (e.g., Microsoft Copilot, ChatGPT, Hugging Face, QuillBot, Canva AI, Perplexity AI, and others).

## Generative AI usage summary (devices)

This widget is available at customer level. It shows the summary of Generative AI usage for a specified time. It includes:
- Total visits
- Total devices with AI usage
- Highest usage device
- Most popular AI solution (visits)

## Generative AI usage summary (customers)

This widget is available at partner level. It shows the summary of Generative AI usage for a specified time. It includes:
- Total visits
- Total customers with AI usage
- Highest usage customer
- Most popular AI solution (visits)

## Generative AI usage details (past 7 days)

This widget shows the detailed information of Generative AI usage for the past 7 days. The table includes columns for:
- Customer
- Machine name
- Domain
- Process
- Visits
- Last visit

## Generative AI usage (visits per device)

This widget is available at customer level. It shows the statistics of Generative AI usage by device as a horizontal bar chart.

## Generative AI usage (visits per customer)

This widget is available at partner level. It shows the statistics of Generative AI usage by customer as a horizontal bar chart.

## Sensitive data transfer trend

This widget shows the dynamics of allowed and blocked sensitive data transfers over a 30-day period. It displays a bar chart with two series:
- **Detected** -- Total number of detected sensitive data transfers
- **Blocked** -- Number of blocked sensitive data transfers

## Top devices with DLP incidents

This widget shows devices with the highest number of sensitive data transfers. The table includes columns for:
- Device name
- Generative AI solution
- Last incident date
- Total incidents
