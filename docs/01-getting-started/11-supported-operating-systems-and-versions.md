# Supported operating systems and versions

## Windows

Unless stated otherwise for a specific feature set, the following Windows versions are supported:

- Windows 7 Service Pack 1 and later
- Windows Server 2008 R2 Service Pack 1 and later

> **Note:** For Windows 7, you must install the following updates from Microsoft before installing the protection agent:
>
> - Windows 7 Extended Security Updates (ESU)
> - KB4474419
> - KB4490628
>
> For more information on the required updates, see the Acronis knowledge base article.
