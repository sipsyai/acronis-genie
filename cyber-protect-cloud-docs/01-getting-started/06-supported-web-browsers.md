# Supported web browsers

The Cyber Protect console uses the TLS 1.2 protocol and supports the following web browsers:

- Google Chrome 109 or later
- Mozilla Firefox 115 or later
- Opera 95 or later (Chromium 109)
- Microsoft Edge 109 or later
- Apple Safari (for Mac OS X) 16 or later, running in the macOS and iOS operating systems

In other web browsers (including Safari browsers running in other operating systems), the user interface might be displayed incorrectly or some functions might be unavailable.

> **Note:** Mobile browsers are not supported.
