# Opening the alert type list

Opening the alert type list

To open the alert type list

Open the Version.
Select Alerts from the menu.



The list includes the following information:



Alert type name



Alert category



Severity
The alert type severity:



Warning
Critical
Error




Comment
The alert type Comment field.




ID

The alert type CTI.



Version
The version number of the CyberApp Version in which the alert type was originally created.





To filter the list
You can filter the list on the Alert type name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.
