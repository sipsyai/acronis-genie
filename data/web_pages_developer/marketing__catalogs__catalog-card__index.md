# Catalog card

Catalog card

For each production-deployed CyberApp, Acronis publishes its catalog card on the integration catalog of the data center(s) on which the CyberApp is deployed.
If the CyberApp is deployed on the EU8 data center, the CyberApp catalog card is also listed on the publicly available Acronis Application Catalog.


A compact version of the catalog card is listed in the catalogs. When the user hovers over the compact version, the full catalog card is displayed.
Generally, the full version of a catalog card includes:



The CyberApp logo.
A brief description of the integration.


Note
The logo and brief description are specified in the About CyberApp section of the CyberApp Description.



A Configure button to activate the integration or to configure the integration, if it is already activated.


Note
This button is not included on the publicly available Acronis Application Catalog.



A Learn more link to display the catalog details page.



Note
For more information about what MSP administrators see, and how they activate a CyberApp integration, see the Management Portal partner administrator guide.
