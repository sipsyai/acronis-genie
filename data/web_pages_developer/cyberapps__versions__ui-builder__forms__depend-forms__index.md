# Dependent forms

Dependent forms

Dependent forms are forms which you create for the root form to create a form group.
The extension points which have UI are:


Custom integration settings tabs.
Main menu items.
Open pop-up workload actions.

Forms in a form group will have some interconnected parent-child relationships
