# UI elements

UI elements
UI builder includes many common UI elements and settings to support a wide range of UX complexity.

UI element groups
The UI elements are grouped by functionality in the UI element menu.






UI element group
UI element



INPUT CONTROLS
Input, Password, Number Picker, Switch, Radio button, Select, Checkbox, Button Group, Date Picker, Time Picker

ACTION CONTROLS
Button, Link, Image

CONTENT ELEMENTS
Text, Header, Alert, Tooltip, Tag, Icon

LAYOUT MANAGEMENT
Divider, Group, Tabs Container

TABLE
Table

CHARTS
Line chart, Pie chart





UI element settings
For UI elements, you can set:


Various properties (ID, label, width, etc.).
For details of the specific properties of each element, see the corresponding UI element details page.


Visibility conditions (whether the element is visible or enabled depending on other elements).
[For some input controls] Validation conditions for user input.
Spacing.
