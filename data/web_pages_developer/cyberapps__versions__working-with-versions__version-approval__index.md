# Sending for approval

Sending for approval
CyberApp Versions must be reviewed and approved by Acronis prior to production deployment.

To send a CyberApp Version for approval

Open the CyberApp’s Version list.

either


Click the CyberApp Version list entry to open the actions and details panel.

The Version state is changed to In review.
