# Viewing statistics

Viewing statistics

Preconditions

MSP has enabled the CyberApp.



Basic flow
The vendor’s ticketing interface must display the information about the CyberApp for MSPs.
For example:

CyberApp configuration status.
Total number of customers.
The number of mapped customers.
The number of generated tickets.
