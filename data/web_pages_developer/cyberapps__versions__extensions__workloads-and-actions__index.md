# Workloads and actions

Workloads and actions
With the Workloads and actions extension point:

The CyberApp can introduce new workloads to Acronis Cyber Protect Cloud.
Your workloads are displayed alongside native Acronis workloads.
Cyber Protect Cloud console users can view the workload information specified in the workload attributes.
Cyber Protect Cloud console users can interact with workloads using your actions.
