# Creating a ledger

Creating a ledger
The MSP wants to be able to create a new ledger in Acronis PSA.

Prerequisite

Partner must have Acronis PSA enabled.


Note
If Acronis PSA is not enabled for this partner, the event should be shown in the pulldown list, but be shown as disabled.



Input fields


Field
Value
Mandatory



Number
Number


External ID
Text string


Description
Text string


Active

One of:
Yes (default)
No










Post conditions

A new ledger object is created in Acronis PSA.
