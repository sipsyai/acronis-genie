# Integration catalogs

Integration catalogs
Integration catalogs are accessible to Acronis partner and customer administrators from the Cyber Protect Cloud platform on their data center (DC).
Integration catalogs list the integrations which are deployed on the DC.

When a user hovers over an integration catalog entry, the catalog card pops up.


Note

Integration catalog listings vary, depending on whether or not vendors deploy their CyberApp on the DC.
When you deploy a CyberApp, you must specify the DCs on which the CyberApp should be deployed.
For more information, see Deploying a CyberApp.
