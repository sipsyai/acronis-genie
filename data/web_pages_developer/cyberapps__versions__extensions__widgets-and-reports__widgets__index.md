# Widgets

Widgets

Widgets visualize data provided by the CyberApp.

Widgets can be one of the following types:



Device status
Show actual data about the devices of the selected types in a pie chart.


Alerts


Active alerts details
Show actual data about the active alerts.



Active alerts summary
Show actual data about the active alerts by alert severity and alert type, as well as the sum of active alerts.



Alerts history
Show historical data about the alerts in a table.



Historical alerts summary
Show historical data about the alerts by alert severity, alert type and time range.
