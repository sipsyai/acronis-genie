# Previewing a Description

Previewing a Description
You can preview the catalog card and catalog detail page which the Description defines without having to test deploy it, and prior to submitting it for production deployment approval by Acronis.

Note
Preview is available only if the Description has all the required content completed.


To preview a CyberApp Description

Open the CyberApp Description list.
Open the Description you want to preview.

Note
You can open the Description in either Edit mode if it is in the Draft state or in Details mode if it is in a different state.


Click .


Select the CARD tab to preview the catalog card.

Note
Currently, the catalog card preview displays the previous version of the catalog card structure. In C25.03, we have implemented a compact catalog, so that more cards would be visible on a single screen. We will update the preview functionality in an upcoming release.


Click Close.
