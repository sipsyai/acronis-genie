# SETTINGS

SETTINGS
In the CyberApp SETTINGS tab, you can:

Rename the CyberApp.
Regenerate a new API key if you have lost the API key credentials.
Manage the connection to Acronis API.
Manage your Acronis OpenID Connect (OIDC) SSO service account.
