# CyberApp enablement

CyberApp enablement
There are two CyberApp Version components which can be involved during CyberApp enablement by an Acronis Partner:


CyberApp enablement form can call Acronis standard enablement callbacks when an Acronis Partner enables a CyberApp.
For more information, see CyberApp enablement form and Enabling the CyberApp.



Customer enablement form can call Acronis standard enablement callbacks when an Acronis Partner enables a CyberApp for their customers.
For more information, see Customer enablement form and Enabling the CyberApp.




Post-enablement

When the CyberApp is enabled, Custom integration settings forms allow Acronis Partners to configure CyberApp features, defaults, and/or behaviors for their tenants. The data specified in custom integration settings forms are submitted to the CyberApp callback handler through custom callbacks.
For more information, see Custom integration settings.
