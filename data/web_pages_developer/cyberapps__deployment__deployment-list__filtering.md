# Filtering the deployment list

Filtering the deployment list
The deployment list can be filtered based on:


Description
To filter by deployment description, start typing in the Search field.
Filtering is dynamic. The list is refined with each character you type.



Type
To filter by deployment type, select an option from the dropdown:



All types (default)
Production
Test



A combination of both
