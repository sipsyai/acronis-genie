# Root forms

Root forms

A root forms is the form which is first presented to CyberApp users when they open an extension point which has UI.
Extension points which have UI are:


Custom integration settings tabs.
Main menu items.
Open pop-up workload actions.


To create a root form

Create a new custom integration settings tabs, main menu item or open pop-up workload action.
Open the UI builder by clicking .


[Optional] Configure the root form.



[Optional] To rename the root form, enter the new name in the Form name field.
By default, the new root form’s name is Root form.


[Optional] Specify a data initializer callback.
[Optional] Specify a refresh callback.
[Optional] Add input parameters.




[Optional] Build the form.
You can do this later, if you prefer.


Click Done.

Important
Clicking Done does not save your work.


Click Save changes to save your new dependent form.
