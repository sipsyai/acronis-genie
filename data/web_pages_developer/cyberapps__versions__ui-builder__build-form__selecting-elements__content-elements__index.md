# CONTENT ELEMENTS

CONTENT ELEMENTS
Content elements are static UI elements which convey information to the user.

Text: A text container, which can contain plain text or text in markdown format.
Headers: Text in a larger font size, typically placed at the top of the form.
Alert: A highly customizable text element, which can be closed by the user. Often used when user attention is required.
Tooltip: An icon is displayed which shows text information about a UI element when the user hovers over it.
Tag: A colored text element. Sometimes also known as a pill, a lozenge, or a chip.
Icon: A customizable icon from an extensive library.
