# Disabling the API connection

Disabling the API connection

Warning

If you disable the connection to the Acronis APIs, theCyberApp connector will not be able to authenticate and API calls will stop working.
See CyberApp connector authentication.



To disable the API connection

Open the CyberApp settings tab.
Turn off the Connection to Acronis API toggle switch.




Click Disable.
