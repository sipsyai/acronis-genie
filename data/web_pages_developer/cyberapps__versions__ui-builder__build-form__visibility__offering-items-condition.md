# Setting offering items condition types

Setting offering items condition types
The Offering items condition checks if the specified Acronis offering item is enabled for the CyberApp user.

To set offering items condition types

From the first User dropdown, select has offering item enabled or has offering item disabled.

From the second User dropdown, select the offering item you want to check for.
Locate the offering item name in the tree structure. You can filter the tree structure by typing a string in the Search field. When you select the required offering item, the offering item code is entered in the dropdown.
Alternatively, if you know the offering item code, you can enter it manually.


Important
If you manually enter an invalid offering item code, the condition will return that the invalid offering item is not enabled.
