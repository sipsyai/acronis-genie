# Configuring

Configuring
Have the option to enable/disable usage and ticket synchronisation while keeping the CyberApp active.

Preconditions

MSP has enabled the CyberApp.



Basic flow
The vendor’s RMM interface must have separate options:

Toggle ticket synchronization on or off.
Toggle usage synchronization on or off.



Post conditions

Tickets and usage synchronization is enabled or disabled.
CyberApp remains active.



API endpoints
None.
