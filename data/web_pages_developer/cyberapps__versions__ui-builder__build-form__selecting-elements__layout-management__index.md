# LAYOUT MANAGEMENT

LAYOUT MANAGEMENT
Layout management elements are used to improve the CyberApp UX by grouping related elements together and separating them from each other.

Divider: A horizontal line, used as a functional or logical divider for aesthetic and UX reasons.
Group: Groups elements together to control the visibility settings of multiple elements as a single group.
Tabs container: A tab folder containing multiple forms.
