# Alerts

Alerts
The MSP wants to see Acronis alert data in the SIEM.

Available data
The MSP can see these Acronis alert data:

Alert severity
Alert body
Alert remediation action
Knowledge base link
Created at
Machine name
Device name
Data center
Customer tenant
Partner name



Basic flow
The MSP can filter alerts:

By priority.
By customer.
By device.



Post conditions
The alert data can be added to any dashboard within the SIEM.


API endpoints

Fetch all alerts.


Note
Specific scenarios and grouping can be achieved with other endpoints found and listed at the same page.
