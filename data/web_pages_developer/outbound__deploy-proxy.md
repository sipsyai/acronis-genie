# Deploying the CyberApp

Deploying the CyberApp

Note

The CyberApp must have a valid CyberApp Description.
All other functionality can be provided outside the CyberApp platform. For example, by accessing the Acronis public APIs from your servers.



To deploy the CyberApp

[Optional, but recommended] Test the CyberApp Description by deploying it to your test environment as a catalog-only deployment.
Request approval of the CyberApp Description.
Publish the CyberApp Description in the Acronis catalogs by creating a catalog-only deployment.

Note

When you deploy catalog-only, the catalog card and details page are published to the integration catalogs, but the Configure button on the catalog card opens the URL you specify in the Action button (Website) field of the CyberApp Description.
This URL should allow the Acronis tenant to enable the integration.




Acronis partner tenants can now find and request to enable the outbound integration from the Acronis integration catalog.
