# Disabling the integration

Disabling the integration
The MSP wants to be able to disable the integration.

Preconditions
The MSP has successfully enabled the integration.


Basic flow
The MSP disables the integration or Workflow.


Post conditions
Integration deactivation is reported via API, and the integration tile in the Application Catalog is set to Available.
