# CyberApp data access scopes

CyberApp data access scopes
To ensure that your CyberApp connector has access to the Acronis Cyber Platform data it requires, you must define the data access scopes in the CyberApp access scopes section of the CyberApp Version.
By default, the connector:


Does not have access to Acronis Cyber Platform data.
Has full access to the extension points that the CyberApp defines.



Note
We recommend that you specify the minimum access scopes necessary for the connector to work.
