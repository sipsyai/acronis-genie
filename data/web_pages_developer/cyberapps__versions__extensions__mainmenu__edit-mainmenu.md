# Editing a menu item

Editing a menu item

Note

If the Version is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Version.
Otherwise, you can only view the Version details.
For more information on Version states, see Version approval process.



To edit a menu item

Open the menu item list.



Select the menu item row.
or
Click  on the menu item row and select Edit.




[Optional] Edit the Basic settings section values.
For more information, see Adding a menu item.


Note
You cannot change an existing main menu item CTI.


[Optional] Click  to modify the menu item form group and click Done when you have finished. For more information, see UI builder
Click Save changes.
