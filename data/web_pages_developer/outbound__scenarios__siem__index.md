# Security Information And Event Management (SIEM)

Security Information And Event Management (SIEM)
SIEM systems offer threat detection, compliance and security incident management through the collection and analysis of security events.
The core capabilities of a typical SIEM system are:

Log event collection and management.
Analysis of log events and other data.
Operational capabilities, such as incident management, dashboards and reporting.
