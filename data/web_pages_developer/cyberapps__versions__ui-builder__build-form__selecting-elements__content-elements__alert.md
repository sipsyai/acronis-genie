# Alert form element

Alert
This element is a colored block with an admonition that requires user attention.
The alert admonition text can contain plain text or text in markdown format.
See examples

Properties



ID
Element identifier.


Width (see examples)


Default
This is for backward compatibility purposes only.



Fit
Sets the width according to the Title and Text property content.



Fill
Uses all of the available width.
For more information, see Element distribution.





Title
The title of the alert, displayed in bold font.



Text
The content of the alert admonition.



Enable markdown
Toggle on to treat the Text field content as markdown syntax.



Note
For more information on markdown syntax, including variable substitution, see UI builder element markdown.




Type
Each type has a different style. (see examples)



Success



Warning



Info



Error





Show icon (default is ON)
Toggle off to hide the alert icon to the left of the alert text.



Enable border (default is OFF)
Toggle on to add a border to the alert block. (see examples)
The border will have a color that corresponds to the alert type.



Hide close icon (default is OFF)
Toggle on to hide the close icon at the upper-right corner of the alert text.



Transparent background (default is OFF)
Toggle on for transparent background color.
