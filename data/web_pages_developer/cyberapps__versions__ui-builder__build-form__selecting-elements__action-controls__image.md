# Image element with option to make clickable to open a link

Image

This element adds an image to the form.
The image can be either static or clickable. If clickable, when the user clicks the image, a URL is opened in a new tab on the browser.


Note
This element is not available in the CyberApp enablement form.


Properties


ID
Element identifier.


The image file.


Note
The image must be a PNG file, with a maximum size of 10 MB, and maximum dimensions of 600x600 px.




Mode
The image element type.



Static

Link
If you select this mode, a field appears for you to enter the URL which will open when a user clicks the image.
