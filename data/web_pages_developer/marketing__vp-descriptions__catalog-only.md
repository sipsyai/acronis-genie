# Deploying a CyberApp to catalog only

Deploying a CyberApp to catalog only

You might want to start promoting the CyberApp to Acronis partners prior to the release.
You can publish a CyberApp Description in the integration catalogs by creating a catalog-only deployment.

If you create a catalog-only deployment, the catalog card and details page are published to the integration catalogs, but the Configure button on the catalog card opens the website you specify in the Action button (Website) field of the CyberApp Description that you deploy.


Note
Catalog-only deployments are applicable for both test and production deployments.


To deploy a CyberApp to catalog only

Create a test deployment or request a production deployment, as normal.
At the appropriate step, select No Version from the CyberApp Version dropdown to only publish the CyberApp Description to the integration catalogs.



Note
For more information on creating a deployment, see Deploying to test and Requesting production deployment.
