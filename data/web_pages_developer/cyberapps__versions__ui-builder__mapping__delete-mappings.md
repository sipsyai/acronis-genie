# Deleting mappings

Deleting mappings

Note

Changing a form element ID deletes any mappings to or from it.
Changing an input parameter ID deletes any mappings to or from it.
Changing the selected callback removes all mappings.



To delete a mapped pair

Click the  button on a mapped pair to delete a single mapping.



To delete all mappings

Select Remove all mappings from the Auto-map dropdown list.
