# Production deployments

Production deployments
For production deployment, you can only deploy  Descriptions and  Versions.


Note
For more information, see the Description approval process section and the Version approval process section.



You can only request a production deployment if the CyberApp has at least one  Description.
If you do not have at least one  Version, you can only request a catalog-only deployment.

Production deployment requests must be approved by Acronis prior to deployment to the specified Data Center(s).
