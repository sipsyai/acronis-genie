# Opening the workload type list

Opening the workload type list

To open the workload type list

Open the Version.
Select Workloads and actions from the menu.



The list includes the following information:



Workload type name



Comment
The workload type Comment field.




CTI

The item CTI.



Version
The version number of the CyberApp Version in which the workload type was originally created.





To filter the list
You can filter the list on the Workload type name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.
