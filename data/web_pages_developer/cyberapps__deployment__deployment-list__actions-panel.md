# Opening a list entry side-panel

Opening a list entry side-panel
The list entry side-panels display deployment request details and available actions.

Note

The details displayed and the available actions depend on the deployment request type and state.
For more information, see the section on production deployments’ actions and details panel and the section on test environment deployments’ details panel.



To open a list entry side-panel

Open the deployment list.
Click a deployment request list entry.
