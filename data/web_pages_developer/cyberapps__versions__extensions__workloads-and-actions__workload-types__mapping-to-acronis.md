# Mapping to Acronis workloads

Mapping to Acronis workloads
If your workload is managed (has an Acronis agent installed), it might already be registered using one of the Acronis workload types.

To match a custom workload type that will be reported with the CyberApp connector, you can specify which workload attribute fields correspond to various the hostname, MAC address, etc.
With these fields, the platform will identify matching workloads, and show them as a single entry in the Cyber Protect console.


Note
For more information, see Creating a workload type.
