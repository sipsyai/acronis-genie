# Acronis catalogs

Acronis catalogs

Acronis promotes integrations to MSP partners in data center-specific integration catalogs and in the publicly accessible Acronis Application Catalog.
Catalog entries consist of two parts:



The catalog card provides a summary of the integration.
The catalog detail page provides more detailed information and marketing content.


Acronis generates the catalog entries using the content of the deployed CyberApp Description, which a Vendor Portal developer or user creates and deploys.
