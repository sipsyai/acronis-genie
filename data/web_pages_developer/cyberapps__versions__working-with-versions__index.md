# Working with Versions

Working with Versions

Your online service can change, and you can decide to add functionality to the CyberApp, so Vendor Portal allows you to create and manage multiple Versions (up to 50 per CyberApp) to reflect evolving functionality.

Each Version is an independently deployable instance of the CyberApp.
So, you can also develop different CyberApp Versions to deploy to specific regional data centers.



Note

Each CyberApp Version must be approved by Acronis before it can be deployed to a production environment.
For more information, see Deploying a CyberApp.
