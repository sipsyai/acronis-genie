# Opening the menu item list

Opening the menu item list

To open the menu item list

Open the Version.
Select Main menu from the menu.




The list includes the following information:


Menu item name


CTI

The main menu item CTI.



Comment
The menu item Comment field.





To filter the list
You can filter the list on the Menu item name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.

Click .
Select or clear column options.
