# General settings

General settings
The General settings tab contains general information of a CyberApp management nature.
It does not contain any information which is displayed in the Acronis catalogs.

To modify General settings content

Note

If the Description is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Description.
Otherwise, you can only view the Description details.
For more information on Description states, see Description approval process.



Open the CyberApp Description.


Modify the general settings.


The Version field defines the Description version number.
This helps you manage multiple CyberApp Descriptions.



Note

Each CyberApp Description version number must be unique.
We recommend incrementing the minor number (for example, 1.1 -> 1.2) for lesser changes, and incrementing the major number (for example, 1.5 -> 2.0) only for more significant changes.





The Comment field helps you identify CyberApp Descriptions in the list, and is used for the search/filter function.


Note

This only appears in the CyberApp Description list.
It is used as the Description column entry.




CyberApp categories define how the CyberApp is listed in the Acronis catalogs.
You must select at least one option. You should select all appropriate categories.



Select languages for localization enables localized versions of the CyberApp Description.
If you select it, the list of localization language options is displayed.
You should select all the languages you intend to include localizations for.



Note

The Acronis catalogs display the CyberApp Description localization file contents according to the default language preference of the Acronis partner.
For this reason, Descriptions can include language localization files.
For more information, see Localizing a Description.






Click Save changes.
