# Adding the request payload

Adding the request payload

Note

If the Version is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Version.
Otherwise, you can only view the Version details.
For more information on Version states, see Version approval process.



To add the callback request payload

Open the callback.
Select the REQUEST tab.
Turn on the Payload toggle switch.
Compose the JSON schema for the payload.


Note

To learn more, see Composing JSON schema for a payload.



Important
If you leave the payload structure empty, the request will not include the payload field.
