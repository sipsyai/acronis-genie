# Importing a Description package

Importing a Description package

To import a Description package

Open the CyberApp.
Select the CYBERAPP DESCRIPTIONS tab.
Click .




Drop the Description package file onto the screen or click choose a file to select the package file from your local disk.
Click Import



Import conflicts
If the Description specified in the Description package has the same version number as a Description in your Description list, an import conflict resolution screen will appear.

To resolve the conflict, the imported Description version number is ‘bumped’ by incrementing the minor version number.
