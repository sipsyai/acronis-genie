# DESCRIPTIONS (CyberApp marketing)

DESCRIPTIONS (CyberApp marketing)

Data center-specific integration catalogs present the CyberApps deployed on each DC to Acronis partners.
Acronis partners can browse available CyberApps and outbound integrations, find out more about the integrations that interest them, and enable the ones they want to use.

CyberApp Descriptions define the information and materials used to generate the CyberApp’s catalog card and details page on the catalogs.
Vendor Portal Developers and Users can create and manage CyberApp Descriptions.


Note
For details, see the Marketing a CyberApp chapter.
