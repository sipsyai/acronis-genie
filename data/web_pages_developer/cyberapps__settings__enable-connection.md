# Enabling the API connection

Enabling the API connection

Note
For more information on the API connection, see CyberApp connector.


To enable the API connection

Open the CyberApp settings tab.
Turn on the Connection to Acronis API toggle switch.


Click Copy.
Paste the information in a file, and store the file somewhere safe.
