# Configuring

Configuring
The MSP wants to have the option to enable/disable usage and ticket synchronisation.

Preconditions

The MSP has enabled the CyberApp.



Basic flow
The vendor’s PSA interface must have separate options:

Toggle ticket synchronization.
Toggle usage synchronization.



Post conditions
As a result, tickets and usage synchronization can be enabled or disabled while keeping the CyberApp active.


API endpoints
None.
