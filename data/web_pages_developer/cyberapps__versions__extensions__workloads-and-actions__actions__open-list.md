# Opening the action list

Opening the action list

To open the action list

Open the Version.
Select Workloads and actions from the menu.
Select the ACTIONS tab.



The list includes the following information:



Action name



Comment
The action Comment field.




CTI

The main menu item CTI.



Version
The version number of the CyberApp Version in which the action was originally created.





To filter the list
You can filter the list on the Action name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.
