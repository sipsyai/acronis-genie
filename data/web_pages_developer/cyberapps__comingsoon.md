# Release notes

Release notes

The following enhancements and features will be introduced in upcoming releases of Vendor Portal:

Custom UI block in customer management/mapping screen.
Ability to report unassigned devices to Acronis, and display and manage alongside devices discovered by Acronis.
Expanded mapping of payload parameters to Acronis parameters for workloads.



C26.02 - March, 2026
This release introduced the following new feature:


Custom callback pagination to populate a table or select element.


The following enhancements and new features were introduced in previous releases of Vendor Portal:

C26.01 - January 28th, 2026


Copy and paste of elements in the UI builder.
Copy and paste of element JSON directly into the UI builder.
Ability to pass parameters from a child form to its parent form.

C25.12 - There was no release
C25.11 - November 28th, 2025

Ability to control the visibility and enablement of CyberApp enablement form elements depending on the CyberApp enablement state.

C25.10 - October, 2025

Alert form element admonition text can be treated as markdown syntax.
Line chart form element.
Pie chart form element.
Set parameter type for button element and table action button On click actions, table On row click actions, and form footer buttons.

C25.09 - There was no release.
C25.08 - September, 2025

Support for font color and size selection in UI builder element markdown.
Radio button UI builder element.
Refresh callback to display progress or update status without active participation from the user.

C25.07 - August, 2025

Auto-mapping of callback payloads in UI builder.
Form Width property for wider display of dependent forms.
Deactivate button automatically added to Settings tab (CyberApp enablement form) of In Use CyberApps.

C25.06 - July, 2025

Localization of Main menu extension point content.
Main menu extension points can display to Acronis partner users, customer users, or both.
Tables support all button types for table action buttons.
Display table action buttons as ellipsis dropdown menu column for each table row.
Buttons dropped into a table row can map callback to the table row model.

C25.05 - June, 2025

Image element with option to make clickable to open a link.

C25.04 - May, 2025

Ability to display a link element as a button.
Improved UI builder group alignment fuctionality.
Improved UI builder element visiblity dependencies.

C25.03 - April, 2025

Update SDK to cover MDR scenarios:


Added OTT support to the CyberApp Sample Components repository.
Updated Acronis Cyber Platform Postman collections to cover MDR scenarios:


Add examples for reading the list of incidents and updating incident state.
Add OTT request example.



Demonstration MDR CyberApp.



Compact format of data center integration catalogs to include more integrations oj a single page.
Improved design of catalog cards.

C25.02 - March, 2025

In UI Builder, link objects and text objects can be dynamically controlled by callbacks.

C25.01 - February, 2025

Selected CyberApp Description fields support basic markdown syntax:


Detail page description
Feature descriptions
Images & videos section description
Additional information section description




C24.12 - January, 2025

New CyberApp-specific overview tab.

C24.11 - December, 2024

New access scopes for tenant provisioning and settings.
Preview for CyberApp Descriptions.
Ability to map from a single data entity to multiple data entities.
Ability to define Input parameters as Primitive data type allows mapping to String, Number, Integer, Boolean or Null form model data types.

C24.10 - November, 2024


UI Builder improvements.

Dependent panels.
UI designer will have the additional option to open dependent forms as a right-side panel instead of a modal window.



Margin and padding settings for UI elements.
Spacing between UI elements will be more flexible.



New icon UI element.
New UI element will let UI designer choose from a large library of icons and set the color of the selected image.



New tab UI element.
Forms will be able to be displayed as folders in a tab element - either horizontally or vertically.



Icon setting for buttons.
Buttons will be able to include an icon as well as or instead of label text.



Table column width settings.
UI designer will be able to set table column widths, and optionally allow CyberApp users to resize columns.







Improved CyberApp import functionality.
Description import.
Version import.
