# Viewing statistics

Viewing statistics

Preconditions

MSP has enabled the integration.



Basic flow
The vendor’s RMM interface must display the information about the integration for MSPs. For example:

Integration configuration status.
Total number of customers.
The number of mapped customers.
The number of generated tickets.
