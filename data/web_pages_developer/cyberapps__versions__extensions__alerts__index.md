# Alerts

Alerts
The Alerts extension point:

Defines alert types to inject alerts created by the CyberApp to Acronis Cyber Platform.
Creates custom alert categories to group alerts of the same nature.
Includes your alert types in your widgets and reports.


Example
Acronis Cyber Protect Cloud displays injected alerts alongside standard Acronis alerts.


Note

For details on how to inject alerts to Acronis Cyber Platform, see the Managing alerts section of the Connector chapter.
To generate real alerts in the Acronis protection console, API users have to pass real data in the payloads.
To facilitate this, Acronis has prepared an Alerts Generator Postman collection, which includes a set of alert examples and their payloads.
