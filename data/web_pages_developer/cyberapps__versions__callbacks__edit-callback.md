# Editing a callback

Editing a callback

Note

If the Version is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Version.
Otherwise, you can only view the Version details.
For more information on Version states, see Version approval process.



Note

Pagination options are not available for version 1.0 callbacks (those created prior to March 2026).
We recommend that you migrate older callbacks to the new callback version. Vendor Portal will do this for you automatically when you request it by clicking on the callback migration banner.



To edit a callback

Open the callback list.



Select the callback.
or
Click  on the callback row and select Edit.



Edit the callback, as required.

For details of callback settings and options, see Adding a callback.
