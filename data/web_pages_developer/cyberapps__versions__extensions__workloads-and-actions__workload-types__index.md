# Workload types

Workload types
Workloads types specify new workloads in the DEVICES section of Acronis Cyber Protect Cloud console, and defines their attributes.

Note
For details on how you can introduce workloads into Acronis Cyber Platform via the CyberApp connector, see The connector.
