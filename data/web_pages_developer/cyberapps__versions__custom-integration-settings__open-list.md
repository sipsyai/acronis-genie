# Opening the custom settings tab list

Opening the custom settings tab list

To open the custom settings tab list

Open the Version.
Select Custom integration settings from the menu.




The list includes the following information:



Item name
The name of the custom integration settings tab.



Note

On the Acronis UI, the tab name will be displayed in ALL CAPS.
So, the Secutiry settings item is displayed as SECURITY SETTINGS in the example on this page.






CTI

The item CTI.



Comment
The item Comment field.





To filter the list
You can filter the list on the Item name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.
