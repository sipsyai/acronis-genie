# Company contact

Company contact

The CyberApp catalog detail page can include the ISV company contact information.
The Company contact content section contains these details.


To add or modify Company contact content

Note

If the Description is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Description.
Otherwise, you can only view the Description details.
For more information on Description states, see Description approval process.



Open the CyberApp Description.
Select Company contact from the menu.


Manage the company contact information:



Company name
The ISV company name.


Note
Required if you add any other company contact details.






Website
The URL of the ISV website.



Email
A contact email for the ISV.



Phone
A contact phone number for the ISV.






Click Save changes.
