# New icon UI element

Icon
This element is a customizable icon, chosen from an extensive, pre-defined library.
See examples

Note
For dynamic icon control based on callback response data, see Defining an icon dynamically.


Properties



ID
Element identifier.


Width


Default
This is for backward compatibility purposes only.



Fit
Sets the width according to Icon size and Icon label properties.



Fill
Uses all of the available width.
For more information, see Element distribution.





Icon name
There are hundreds of icons available to choose from.



[Optional] Icon size
There are 6 sizes available. (see examples)
If you do not select a value for Icon size, the icon is displayed with its original size. This is indicated by the number at the end of the icon name.



[Optional] Icon color
There are 42 colors available. (see examples)



Note
This option is only available if you selected a monochrome icon on the previous step. Colored icons are displayed with their original color(s).




[Optional] Icon label
Text displayed to the right of the icon.
