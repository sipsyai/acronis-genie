# Custom integration settings

Custom integration settings

In Custom integration settings, you can define custom settings tabs, which are displayed on the Acronis Cyber Platform INTEGRATIONS page when the CyberApp is activated by an Acronis partner.
These tabs can allow partners to configure CyberApp features, defaults, and/or behaviors for their tenants, for example.

The data specified in custom settings tabs are submitted to the callback handler using custom callbacks.

Example
DemoCyberApp has a custom settings tab called SECURITY SETTINGS, where Acronis MSPs control password security policies.
