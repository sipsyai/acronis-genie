# Opening the callback list

Opening the callback list
The callback list displays all the custom callbacks for the CyberApp Version you are currently working on.

To open the callback list

Open the Version.
Select Callbacks from the menu.



The list includes the following information:



Callback name



Type
Currently, the callback list only displays custom callbacks.



Comment
The callback Comment field.




CTI

The callback CTI.



Version
The version number of the CyberApp Version in which the callback was originally created.





To filter the list
You can filter the list on the Callback name column.

To filter the list, start typing a string in the Search field.
Filtering is dynamic: the list is refined with each character you type.

Click .
