# Sending for approval

Sending for approval
CyberApp Descriptions must be reviewed and approved by Acronis before you can publish them.

Note

The  option is only available if the Description is valid and the state is Draft.
If your Description is not valid, add the required elements before proceeding.



To send a Description for approval

Open the Description list.

either


Click the Description list entry to open the actions and details panel.




Click  in the available actions bar.


or


Click the  button on the Description list entry.
Select  from the dropdown.





[Optional] Enter a comment.
The comment appears in the Reason for state change field of the Description actions and details panel.


Click Send for approval.

The Description state changes to In review.
