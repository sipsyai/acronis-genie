# Building a form

Building a form

To build a form

Open the form.
Drag an element from the element menu and drop it on the canvas.

Note

Form elements are positioned vertically in a single column.
Drop elements above or below another element in the form.
To position elements horizontally, add a Group element to the form, set the Mode to Row, and drop elements on it.



Enter the element id in the ID field.

Set the other element properties.
For more information on the available properties, see the element type details page.


[Optional] Create dependencies for the element’s visibility.
[Optional for certain input elements] Set the element validation criteria.

[Optional] Change the default spacing of the element.


Repeat steps 2 to 6 for all necessary form elements.

Note

To copy and paste an element, use the Ctrl+C and Ctrl+V keyboard shortcuts (Command-C and Command-V on macOS). For details, see Copying and pasting elements.
To delete an element, select it and either use the Ctrl+X keyboard shortcut (Command-X on macOS) or click .
If you use Ctrl+X (Command-X on macOS), you can undo the delete using Ctrl+Z (Command-Z on macOS), but only if you do this prior to any other change to the form.
To see details about an element, select it and click .



[Optional] To preview the form, click .
When you have finished building the form, click Done.

Important
Clicking Done does not save your work.


Click Save changes to save your form/form group.
