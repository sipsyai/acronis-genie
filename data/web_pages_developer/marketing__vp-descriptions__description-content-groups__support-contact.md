# Support contact

Support contact

The CyberApp catalog detail page includes support contact information for the CyberApp.
The Support contact content section contains these details.


To add or modify Support contact content

Note

If the Description is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Description.
Otherwise, you can only view the Description details.
For more information on Description states, see Description approval process.



Open the CyberApp Description.
Select Support contact from the menu.



(Required) Provide the CyberApp support Company name


Manage the support contact information:



Website
The URL of the CyberApp support webpage.



Email
A contact email for CyberApp support.



Phone
A contact phone number for CyberApp support.




Note
We recommend that you provide at least one way for CyberApp users to contact support.



Click Save changes.
