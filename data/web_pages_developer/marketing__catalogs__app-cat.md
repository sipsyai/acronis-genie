# The Application Catalog

The Application Catalog

The Application Catalog is publicly available on the Acronis website.
It displays a list of CyberApp and outbound integration ‘proxy’ CyberApps whose Descriptions are deployed on the Acronis marketplace data center.
The entries are for reference only. There is no Configure button on the Acronis website catalog cards.


Note
The Application Catalog displays CyberApp Descriptions deployed to the marketplace data center.
If you deploy to any other data centers without deploying to the marketplace data center, no entry is published in the Application Catalog.
For more information, see Requesting production deployment
