# Features

Features

The CyberApp catalog detail page can include a two-column list of CyberApp features.
The Features content section contains the details of each feature, and orders the list.


Note
Acronis recommends providing at least four features.


To add or modify Features content

Note

If the Description is in the Draft state, and you have the appropriate Vendor Portal account type, you can open and edit the Description.
Otherwise, you can only view the Description details.
For more information on Description states, see Description approval process.



Open the CyberApp Description.
Select Features from the menu.


Manage the features list.

Select  to add a feature to the list.
