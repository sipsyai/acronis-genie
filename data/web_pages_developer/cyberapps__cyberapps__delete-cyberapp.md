# Deleting a CyberApp

Deleting a CyberApp
To delete a CyberApp, you must first delete any Descriptions and delete any Versions.

To delete a CyberApp

Open the CyberApp list in the list view.
Click the  and select Delete.

Note
Delete is only available if the CyberApp does not have any Descriptions or Versions.

A verification window appears.

Click Delete to verify.
