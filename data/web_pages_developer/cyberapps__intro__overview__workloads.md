# Alerts, workloads, and workload actions

Alerts, workloads, and workload actions

Alerts

CyberApp Connector can manage alerts in the Acronis platform.
For more information, see Managing alerts and the Alert Manager API reference.

The alerts extension point allows you to add CyberApp-specific alert types to the existing Acronis alert types.
For more information, see Alerts.



Workloads

CyberApp Connector can manage workloads in the Acronis Cyber Protect Cloud.
For more information, see Managing workloads.

The workloads extension point allows you to add CyberApp-specific workload types to the existing Acronis workload types.
Workloads types specify new workloads in the DEVICES section of Acronis Cyber Protect Cloud console, and defines their attributes.
For more information, see Workloads and actions.



Workload actions

Workload actions appear as entries in the actions list of the DEVICES section of the Acronis Cyber Protect Cloud console.
The user can perform the actions on either native Acronis workload types, CyberApp-specific workload types, or a combination of both.
For more information, see Actions.
