# Demo CyberApp

Demo CyberApp
SayHi is a very simple CyberApp in the style of the classic “Hello, World!” demonstration programs. In order to keep the demo simple, SayHi does not perform any partner or customer enablement, no callback handler or custom callbacks are required, and no CyberApp connector is required either.

SayHi adds a menu item to the Acronis Cyber Platform MANAGEMENT menu.
When a user clicks the menu item, a CyberApp form is displayed with a select element, an input element, and two disabled buttons.
When the user selects an option from the select element, the two action buttons are enabled.
One button opens a form in pop-up mode. The form displays the chosen greeting and optional additional message.
The other button opens the same form in sidebar mode.
