# Creating your first CyberApp

Creating your first CyberApp

Important

Only Vendor Portal developers can create a CyberApp.
For more information about Vendor Portal account types, see Vendor Portal.



To create your first CyberApp

[If you are not already in Vendor Portal] Access Vendor Portal.
Select MY CYBERAPPS from the main menu.


Click Create CyberApp.


Enter a name for the CyberApp.
[Optional] Edit the CyberApp code.


Note

The CyberApp code must be unique. It consists of two parts:

Vendor Id
Unique to your organization. Cannot be modified.



CyberApp Id
CyberApp name (in lowercase) is the default.








Click the  icon.



Change the CyberApp Id part of the CyberApp code.
To revert to the default, click Use default code.




[Optional] To access the Acronis API, turn on the Connection to Acronis API toggle switch.

Important
A connection to Acronis API is required to allow the CyberApp to push/pull data to/from the Acronis platform.



Click Create.
If you did not turn on the Connection to Acronis API toggle switch, the CyberApp is created.
If you turned on the Connection to Acronis API toggle switch, Client ID and Secret credentials are generated.



Important

Client ID and Secret are credentials that are critical for the authentication of the CyberApp and API calls.
You must copy and store them in a safe place.
This information cannot be retrieved if lost.



Click Copy.
Paste the information in a file, and store the file somewhere safe.
Click Close.




You have created your first CyberApp.
For more information, see Managing CyberApps.

You can now open the CyberApp and add marketing materials or start developing the CyberApp functionality.
