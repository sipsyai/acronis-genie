            Hi Lawrence
I have previously seen the warning "Failed to detect OS", this happens when the machine you are backing up is using LVM disks in most cases I found this on specifically Ubuntu OS.
If you are on the latest build it should show as an information message instead of a warning believe this was from version 28037
This message can be safely ignored as it does not have any impact on the backup,
