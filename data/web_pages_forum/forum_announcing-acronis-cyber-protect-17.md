            Hello! 
We are glad to announce the Acronis Cyber Protect 17.
Acronis Cyber Protect delivers a unified cyber resilience platform that combines data protection, cybersecurity, and endpoint management in a single, integrated solution.
It’s designed for organizations that require data sovereignty and compliance, supporting both fully air-gapped deployments and environments with internet connectivity.
 
? What’s New in Acronis Cyber Protect 17
 
? On-Premises Deployment
 
? Agentless Backup for Nutanix AHV
Protect Nutanix environments without agents — fast, simple, and fully integrated.Key Benefits: Native Nutanix integration, cross-platform recovery, flexible storage options.Scenarios: Backup and recovery of Nutanix AHV VMs, migration to/from other platforms.Licensing: All editions.
? Agentless Backup for Proxmox VE
Zero-touch protection for Proxmox VE environments with centralized management.Key Benefits: No per-VM agents, unified console, cross-platform recovery.Scenarios: Backup of Proxmox VMs, restore physical machines as Proxmox VMs.Licensing: All editions.
? Role-Based Access Control (RBAC)
Assign roles with precision to minimize risk and human error.Key Benefits: Dedicated roles (Backup Admin, Restore Operator, Security Admin) for compliance and security.Scenarios: Role-based permissions for backup, restore, and security tasks.Licensing: All editions.
? Device Sense™ Discovery
Detect every device — even unmanaged ones — to close protection gaps.Key Benefits: Instant discovery, network visibility, stronger security posture.Scenarios: Network-wide endpoint detection within registered segments.Licensing: All editions.
? Software & Hardware Inventory
Gain real-time visibility into your IT assets.Key Benefits: Automated inventory scans, update planning, compliance tracking.Scenarios: Asset management, patch planning, and license control.Requirements: Agent installation.Licensing: Standard & Advanced editions.
?️ Support for New Operating Systems
Protect the latest environments from day one.Supported: Windows Server 2022/2025 Essentials, Ubuntu 24.10, Fedora 39–41, Oracle Linux 9.5, CloudLinux 9.5, AlmaLinux 9.5, Rocky Linux 9.4/9.5, Red Hat Enterprise Linux 9.5.Licensing: All editions.
 
☁️ Cloud Deployment
 
? Support for macOS 26 Tahoe
Full cyber protection for the newest macOS release.Scenarios: Backup and protection of macOS 26 Tahoe systems.Licensing: All editions.
? Detection & Response: Automated Agent Uninstallation Protection
Active Protection now automatically blocks unauthorized attempts to uninstall the Acronis Agent — defending against ransomware tampering.Key Benefits: Prevents unauthorized removal, maintains protection integrity.Scenarios:
• Agent uninstallation and updates blocked by default.
• Administrators can enable a maintenance period (1 hour – 7 days) to allow updates.Licensing: Advanced edition.
 
⚙️ Installation Notes
 
? Upgrading the Cyber Protection Agent
You can upgrade directly to Acronis Cyber Protect 17 from any version 15 or 16.
?️ Upgrading the Management Server

Direct upgrade supported from Acronis Cyber Protect 16 → 17.


If running version 15, upgrade to 16 first, then to 17.

Important:
• An active subscription license or support maintenance agreement is required.
• Suspend BitLocker before upgrading agents to prevent recovery-key prompts.
• If using immutable storage, review the Account Server certificate update steps.
• For Virtual Host/Universal license use inside VMs (agent-based backups), see the related Knowledge Base article.
