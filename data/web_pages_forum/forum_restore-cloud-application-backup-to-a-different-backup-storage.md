Unfortunately, that’s not possible.
If you perform C2C backups (cloud to cloud), you can only recover them directly to OneDrive.
Since cloud-to-cloud backups do not have any local agents installed, there is no way to connect to a local machine and recover the backups there.
Please check the available options:
Recovering OneDrive Files
Additional Recovery Options
