            Hello, Johnathan!
I apologize that no one replied,
 
In all cases that I have had, this was caused by external provisioning systems, which stores another value of the quota.
Please check if you or your service provider uses those systems.
If you do, set the quota through the provisioning system - it will probably require a new service plan (or however different sets of quotas are called in any given example).
 
If you would have any questions, please let me know the name that you use to log in to cloud.acronis.com, we will be able to see which integration system is used.
