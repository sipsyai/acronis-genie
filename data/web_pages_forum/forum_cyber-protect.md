            Hello!
Welcome to the forum.
A backup of 250GB (assuming it’s the first one) will take several hours since it’s a full backup.
You should review the Backup Storage tab to see the actual size being occupied.
When browsing the vaults, you will be prompted to provide the encryption keys to access the recovery points.
Also, ensure the backup is complete. You need to verify the size of the backup once it has finished. The size displayed is accurate and valid only after the backup is fully completed.
