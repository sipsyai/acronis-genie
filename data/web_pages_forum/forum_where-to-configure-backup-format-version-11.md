            Hello Bruce,
Thanks for participating.
Please refer to this user guide: https://www.acronis.com/en-us/support/documentation/AcronisCyberProtect_16/#backup-format-acp.html#kanchor60
As stated, it is only available for protection plans that use the legacy backup format version 11. You should export the plans from ACP15 to ACP16 in order to have that option available.
If you create a new plan in ACP16, that option won't appear.
