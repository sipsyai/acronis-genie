            Hello Shim Win,
Thank you for your question, I'll be glad to explain.
You can see the size of the full archive in the Backups tab of Backup Management Console, but it will not show you the single daily backups size.
In order to see each backup size, you should login to the Web Restore Console and browse down to the archive you are interested in:
Select a machine
Go to Recovery
More ways to Recover
Download Files
Login with your end user credentials
Select the machine
Select the Archive (it does not show backu plan name, so you should select based on it's size and type which you can check priorly in Backups tab)
There you will see all the backup slices with dates and sizes.
I've attached a screenshot with these steps, they should be sufficient.
Let me know if you have additional questions.
