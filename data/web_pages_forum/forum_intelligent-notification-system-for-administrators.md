            Hi Christian
Acronis Backup Cloud currently have the following alerts:
Backup notifications:
Off: Which does not send any notifications
Less: Which sends notifications about errors
More: Which sends notifications about errors and warnings
All: Send all type of backup notifications
Then there is the option to send daily recap about active alerts, business notifications and then lastly which was added recently is within the actual backup set you can enable an alert which will notify you if no successful backups for a specified number of consecutive days has been reached.
The Acronis Backup Cloud backup management console also has the ability to send out reports:
Default is "Scheduled" which send a monthly summary report
Custom:Can generate a custom report based on personal requirements (Such as daily statistics etc)
I hope this answer helps you.
