Hello Carlos,
If you don't have permissions over the VMware host, you must install an agent inside each VM. Note that the user of that VM must have admin roles enabled, and those VMs must have all the network settings configured properly to connect with the datacenter.
You can use Acronis Cyber Protect 16, which allows you to use it on-premise or in the cloud console: https://www.acronis.com/en-eu/products/cyber-protect/purchasing/
You can also register for a trial; you have 30 days to use it and check if it fulfills your expectations: https://www.acronis.com/en-eu/products/cyber-protect/trial/
Note that replication is available only for VMware ESXi virtual machines, so if you can't install the ESXi agent, you can't create a replica. In that case, only regular backups can be created: https://www.acronis.com/en-us/support/documentation/AcronisCyberProtect_16/#replication-of-virtual-machines-acp.html#kanchor168
