            Dear Community,
We are thrilled to announce the release of Acronis Cyber Protect 16! ?
This release is specifically designed for our valued existing subscribers of Acronis Cyber Protect, as well as new customers eager to explore its array of benefits.
 
Here's a quick overview of the exciting new features:
? Introduction of a centralized dashboard (compatible with on-premises deployments)
? Implementation of immutable storage (supported in both on-premises and cloud deployments)
? Streamlined one-click recovery process
? Automated modular agent deployment and uninstallation
? Integration of Agent for Synology
? Support for Tape LTO-9 format
? Added compatibility for MacOS on ARM CPU
? Inclusion of support for Microsoft SQL Server 2022 as the database for the Acronis Management Server
? Enhanced recovery capabilities for workloads protected by BitLocker
? Option to disable local folder browsing
 
For more detailed information, please refer to the release notes available here.
 
You can directly download the agent from here or through our Acronis account portal.
