            Hello!
The EWS (Exchange Web Services) address for Microsoft 365 (formerly Office 365) is typically:
https://outlook.office365.com/EWS/Exchange.asmx
Key Points:
This is the standard endpoint for Exchange Online in Microsoft 365.
Ensure that the account you're using for backup has the necessary permissions, such as impersonation rights, to access the mailboxes.
Verify that the EWS API is enabled for the organization (it is enabled by default unless restricted by an admin).
If you're using Acronis Cyber Protect, it will usually handle the configuration automatically when you connect your Microsoft 365 organization for backup. Ensure you've followed the product's setup guidelines for mailbox backups.
