            ? Acronis Cyber Protect 16 Update 4 Released
We’re excited to inform you that Acronis Cyber Protect 16 Update 4 (Build 39929) was released.
? What's New
? On-Premises Deployment:

Option to install Agent for Windows without Active Protection components


Ability to join a machine to a domain after recovery


Migration from CentOS 7 to Rocky Linux for Acronis Management Server (Docker container image)

☁️ Cloud Deployment:

Agentless backup support for Nutanix AHV virtual machines


EDR support for Linux workloads


Enhancements to the Startup Recovery Manager


Support for WinRE-based restore with reboot

? All Deployments:

Expanded Operating Systems Support:
	– Windows Server 2025
	– Linux kernels 6.10, 6.11
	– Hyper-V on Windows Server 2025


Improved hardware support with Linux-based bootable media ( The Linux-based bootable media now uses kernel version 6.12, enabling out-of-the-box support for a broader range of hardware ).

? The full release notes are available here
Feel free to share your feedback or questions in the comments below!
