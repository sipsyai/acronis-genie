thank you for your posting! The error message in this case is incorrect, we've already passed it to the developers. Regarding the issue with Agent registration, I'd start with checking the connection between the appliance and the Acronis Management Server per https://kb.acronis.com/content/63130 The Agent needs to have access to the hostnames/IP addresses and ports listed in this article https://kb.acronis.com/content/47189 
Make sure, you are on the latest build.
