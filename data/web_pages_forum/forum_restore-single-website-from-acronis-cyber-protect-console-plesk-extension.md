            Hello!
Welcome to the forum.
Unfortunately, that’s not possible with the Plesk integration. Just granular recovery of websites is possible.
Please refer to the following user guide:
With the Acronis Backup extension for Plesk, a Plesk administrator can:
Back up an entire Plesk server locally, to a network folder, or to cloud storage
Recover the entire server, including all websites and databases
Perform granular recovery of websites, individual files, mailboxes, or databases
Enable self-service recovery for Plesk customers and resellers
For more details, check the user guide here.
