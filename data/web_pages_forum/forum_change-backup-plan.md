            I'm using Acronis cloud backup to backup a couple of VM's. In my Acronis cloud console I have created some backup protection plans which contain properties like start time, retention, days to backup etc.
Suppose I have VM1 on Plan1 and I want to change the plan on VM1 so that it runs with Plan2 instead of Plan1, how do I do this? I do not want to edit Plan1, I want to switch over to Plan2 on a VM.

Simply disable/revoke Plan 1 on VM1 and create and apply Plan 2 on VM1. 
This way you will not Edit Plan 1 and will have successfully changed VM1 to Plan 2.
