            Good day Moris,
This is most definitely possible, when running a file/folder level restore:
Under "recover to" you will see "path" with two options:
1. Original path - This will restore the file/folder to the original path it was on
2. Custom location - This will allow you to restore the file/folder to any other location you require, when selecting custom location you are then again prompted with two options:
1. Local folder - This being the local devices on the machine
2. Network folder - This allows you to restore to any other location via for example SMB or CIFS
Hope this helps,
