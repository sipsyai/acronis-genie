            In the Backup management console, click Recover, then select the location and the recovery point. For a file-level archive you have one option - recover files. For a disk-level archive you have two options - recover files and recover machine. Click Recover files and select the data you want to recover.             By local storage you mean a local/network folder, not an own Acronis Storage (another Acronis solution) as a replacement for the cloud storage in Acronis data centers? The topic starter was asking about Acronis Storage, so I thought you had Acronis Storage as well.
The recovery method is basically the same for both cloud and local storage: you click Recovery, select location and recovery point and click Recover files. The only difference between restoring files from a local storage and a Cloud storage is that you can download files from the Cloud or directly restore to the original machine from a local/network folder. Please see the detailed description here: https://dl.managed-protection.com/u/baas/help/4/admin/en-US/index.html#…
