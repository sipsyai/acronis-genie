            Hello Peter,
Welcome to Acronis forums!
Acronis Cyber Fit Score is a standalone utility that evaluates the security state and identifies security gaps in the IT environment and gives recommendations for improvements.
The CyberFit score functionality is supported on:
Windows 7 (first version) and later versions
Windows Server 2008 R2 and later versions
After you run this utility it calculates your score in the same small program window where you launched it (see as in my example):

Then it redirects to Acronis page with the details of your security assessment and recommendations on how to improve the score.
Please read more details on this tool's functionality at: https://www.acronis.com/en-us/support/documentation/CyberProtectionService/index.html#47817.html
