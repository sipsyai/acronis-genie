            Hello,
we are evaluating the Acronis Disaster Recovery Cloud solution but at the moment I think it lacks of an important feature about networking: because the VPN tunnel just creates an L2 extension there isn't a tool to manage in cloud the communications between VMs which belong to different VLANs. The Acronis DR solution seems able to protect only the VMs in the same subnet where the VPN appliance is placed. Because is really common to have many subnets in the same organization I would like to know if Acronis has in roadmap to cover this possibility and the expected timeline.
Regards,
