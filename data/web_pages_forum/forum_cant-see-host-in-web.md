            Hello Dmitry,
 
Thanks for posting on our Forum.
As far as I understand you installed the Agent for Hyper-V but the virtual machines that are running on the host are not shown in the console.
The issue's root cause can depend on different components so without the system report it is hard to investigate.
You can check if the machines were cloned from one server to another with saving of machine Id.
There is workaround for this case:http://www.amixa.com/blog/2014/06/30/quick-change-of-hyper-v-uid-for-register-in-place-vm/
 
If this does not help, please submit a ticket per our Service Provider template
