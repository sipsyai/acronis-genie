            Dear Acronis Customers,
Please be aware that we have released backup extension for Windows Azure Pack.
The extension integrates Windows Azure Pack with Acronis Backup Cloud and enables service providers to offer Acronis Backup Cloud as an add-on to IaaS offering. Using the service tenants can back up both cloud virtual machines and on-premise infrastructure.
Currently this extension is provided per request. Product team is working with Microsoft team on certification to make it publicly available through Microsoft Web Platform Installer.
Release notes are attached.
Thank you,
