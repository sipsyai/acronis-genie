            My observations and first issues (upgrade from 16.0.37482 to 16.3.39376).
1. "External IDs conflict for >1 credential objects" error when trying to refresh list of backups in "Backup storage" section. Support answer: "The issue is known issue and yes it will be fixed in the update 4, however update 4 is planned to release in the beginning of Q2". OK, let's wait.
2. Some of backup locations within one backup plan lost access (Message: No access/permissions). In most cases resolved by remove and reenter the same location in backup plan but some still has error.
2. Only 30% of license keys available to upgrade to v16 (in 16.0.37482 all agents worked fine on v15 keys). Problem resolved by support.
3. Bootable Media "Windows PE 64 bit" does not automatically register on management server, not assign licence key and no Backup option available even manual registered and assigned the key. Bootable Media "Linux based" work fine but this is not a solution.
