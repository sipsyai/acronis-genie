            Salut guillaume
Avec Acronis Backup Cloud, le cryptage est défini à partir du niveau du jeu de sauvegarde et ne peut donc pas être modifié. Si la clé de cryptage est perdue / oubliée, toutes les données sauvegardées seront malheureusement perdues.
De plus, Acronis Backup Cloud ne prend pas en charge le blocage des ports USB. Le seul message que l'utilisateur recevra est que si Acronis Backup Agent est installé sur la machine et que l'outil de surveillance est installé, un utilisateur verra quand la sauvegarde est en cours d'exécution, etc.
En gros, Acronis Backup Cloud est une solution de sauvegarde et de récupération avancée avec une protection active intégrée pour protéger le périphérique, les lecteurs USB qui lui sont connectés ainsi que les lecteurs réseau des menaces telles que Ransomware.
J'espère que ceci vous aide.
