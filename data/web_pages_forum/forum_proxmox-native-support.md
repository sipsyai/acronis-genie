Hello James,
Thank you for your feedback.
The status of this feature remains without an ETA. Please note that this thread is intended for the cloud version of the product.
Upon reviewing your account, I see that you are using the on-premise version.
For Proxmox, Acronis Cyber Protect 15 and 16 support it as a KVM-based hypervisor in agent-based mode. This means you can protect the VMs by installing an agent inside each VM. However, agent-less backup and paravirtual VMs are not supported.
To protect Proxmox hosts, you will need one Advanced Virtual Host license per host, which allows you to protect an unlimited number of VMs on that host.
