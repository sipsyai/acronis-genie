            Hi Jørgen!
 
If by Teams you mean Sharepoint Group (team) sites, Acronis Data Cloud is able to back them up, they will appear under "All site collections" or "All groups" in the Microsoft Office 365 tab.
 
If you mean Office 365 Teams (https://products.office.com/en-us/microsoft-teams/group-chat-software), this is not supported yet.
 
If some of the groups are not discovered, please submit a support request that would include screenshot from SharePoint web overview depicting the missing sites.
