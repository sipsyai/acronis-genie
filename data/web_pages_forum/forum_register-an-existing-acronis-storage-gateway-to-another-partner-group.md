            Hello Eric.
This is Dmitry from the Acronis Service Providers Support Team.
Thank you for your question.
In case if the partner doesn't planning to move the backups under the new group and only wants to relocate existing storage to the new group: it will be enough to re-register the storage using the admin account of a new group.
If he wants to move the storage with accounts and with the existing backups for these accounts - it is not possible yet. This feature is on the development stage and should be available preliminary in Acronis Backup Cloud 8.0
