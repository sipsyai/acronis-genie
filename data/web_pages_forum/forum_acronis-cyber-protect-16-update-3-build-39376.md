            What's New in Acronis Cyber Protect 16 Update 3 ?
Here’s a rundown of the exciting new features and updates in Acronis Cyber Protect 16 Update 3:
?️ All Deployments
Support for New OS Versions
macOS 15 Sequoia support added.

? On-Premises Deployment

Plugin for VMware vSphere Client
Integration with VMware vSphere Client enables VMware users to perform self-service agentless VM backups and recoveries directly from the vSphere Client web console UI.
Supported for HTML5-based vSphere Clients (versions 7.0.1 and higher).


Licensing:
Available in all editions.

☁️ Cloud Deployment

Antimalware Scan of Microsoft 365 Mailbox Backups
Key Benefits:
Ensures malware-free email recovery from Microsoft 365 mailbox backups.
Adds a second layer of defense if email security solutions fail to detect malicious content.
Extends malware scanning functionality to Microsoft 365 mailbox backups at no additional cost.

Licensing:
Requires Acronis Cyber Protect Backup Advanced license.



Search in Microsoft 365 and Google Workspace Encrypted Backups
Enables the ability to search content within encrypted archives, saving time when the exact file location is unknown.
Licensing:
Requires Acronis Cyber Protect Backup Advanced license.


?️ Cyber Protection Agent Uninstallation Prevention

This feature replaces the password protection of the Cyber Protect Agent, providing stronger defense against ransomware that attempts to uninstall security software before encrypting data.


Starting with this release, uninstallation and updates of the Cyber Protect Agent are blocked by default. They can only occur during maintenance periods (from 1 hour to 7 days).


Licensing:
Available in all licenses.

? Virtualization Protection: Added Support for Hyper-V Generation 2 Virtual Machines

Support added for cross-platform restores to Hyper-V Generation 2 VMs from EFI-based systems.
Scenarios Supported:
Restoring backups as Hyper-V Gen 2 VMs if the source system is EFI-based.
Cross-platform restores of Linux EFI-based systems to Hyper-V.
Restoring systems with more than 4 partitions (EFI + GPT).
Running as VM (Instant Restore).



Licensing:
Available in all licenses.

Stay tuned for more updates, and as always, feel free to reach out if you need assistance! ?
