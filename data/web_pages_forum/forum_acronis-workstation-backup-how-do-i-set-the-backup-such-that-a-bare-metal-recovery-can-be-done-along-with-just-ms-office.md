            Hello!
Welcome to the forum.
You need to create bootable media and then manually select the files from there:https://www.acronis.com/en-us/support/documentation/CyberProtectionService/#creating-bootable-media.html#kanchor1415
By default, the system selects all fixed disks for backup. To change what will be backed up, click Items to back up and select the disks or partitions you want.
If you want to back up files or folders instead of disks, switch to Files in Data to back up.
If you have any questions, please contact our support team or your service provider if you are not entitled to direct support.
