Thankyou for responding.
So firstly we are an MSP as the core of our business. Therefore:
1. Integration between Acronis Cloud sites/companies and salesforce companies
2. Integration between Acronis Cloud alerts and Salesforce Service Cloud Cases
3. Integration between Acronis Cloud sites/companies usage data in salesforce
4. Integration between Acronis Cloud sites backup policies and salesforce
