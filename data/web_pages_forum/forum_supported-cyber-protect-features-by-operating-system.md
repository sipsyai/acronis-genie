            Perhaps it wasn't clear in the post. Now with more detail:
For Cyber Protect version 16, where is the table of Supported Cyber Protect features by operating system that is in the Cyber Protect version 15 User Guide. In the PDF version of the Cyber Protect Version 15 User Guide, Update 6, dated 11/12/2024, the table of Supported Cyber Protect features by operating system is on pages 17 - 20. In the online version of the Cyber Protect Version 15 User Guide, dated 11/12/2024, the table of Supported Cyber Protect features by operating system can be found by clicking in Contents on Acronis Cyber Protect 15 editions, then clicking on Supported Cyber Protect features by Operating System.
The first link referenced in the above post:
https://care.acronis.com/s/article/73376-Acronis-Cyber-Protect-16-On-premises-and-Cloud-features?language=en_US
navigates to a Knowledge Base article# 000016278, titled Acronis Cyber Protect 16: On-premises and Cloud features, which as it states is a table of the feature difference between Cloud and on-premises deployment. That table has NO reference to Operation System at all.
The second link referenced in the above post:
https://www.acronis.com/en-us/support/documentation/AcronisCyberProtect_16/#supported-cyber-protect-features-by-operating-system.html
navigates to the online version of the Cyber Protect Version 16 User Guide, update 3, dated 12/23/2024, Software Requirements. The Software Requirements section of the document does not have the table Supported Cyber Protect features by operating system. Even though the link has the words "supported-cyber-protect-features-by-operating-system" in the link, the link does NOT navigate to a location in the online version of the Cyber Protect Version 16 User Guide with that information.
The third link reference in the above post:
https://www.acronis.com/en-sg/resource-center/resource/acronis-cyber-protect-editions-comparison-including-cloud-deployment/
navigates to a Datasheet which is detailing Cyber Protect Version 16 editions, i.e. Standard, Advanced, Backup Advanced verses features. Again, does not reference Operating Systems.
Where did the second link in the above post come from?
The question is NOT answered.
