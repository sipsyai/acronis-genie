Dear Acronis,
currently, the backup agent uses multiple ports. Some of them are non-standard.
This is a problem for us because we have government customers. They have a very restricted firewall policy and do not allow to open port 44445 outgoing. Some time ago, I have sent them a list of IP addresses to open the ports to. But after a month or two, you have added more IP addresses and the backups did fail.
It is clear to me that you as a cloud provider can not specify a list of IP addresses which will not change.
So I kindly ask you to change all communication ports to 443 and use TLS. Port 443 is open even on restrictive firewalls. It would not only be a benefit for partners with government or enterprise customers. Even the firewall admin will open the ports, it is still effort for the customer because he has to involve his IT supporter. With port 443, your partners could install the agent and it would "just work".
Thank you for considering my request!
