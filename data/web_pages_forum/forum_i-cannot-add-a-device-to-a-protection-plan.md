            Good day,
If you are running the trial through a reseller or distributor you will need to reach out to them to amend the allocated licenses, you will need to specify what you are trying to backup as the workstation backup does also come with advanced features
If you have an Acronis account manager who setup the trial for you, you will need to do the same as above as they will need to amend the license allocation so that you can test what is needed,
If you got the trial from the website then you will need to reach out to support to correct this for you,
If you would like to just test the workstation backup as is with no advanced features then you can disable the advanced features and the issue should go away by itself,
