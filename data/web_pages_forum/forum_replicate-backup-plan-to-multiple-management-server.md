            Hello!
There is no way to make the program move backups from one location to another machine automatically.
You can either move backups between storage nodes by following this article:https://care.acronis.com/s/article/61149-Acronis-Backup-12-5-How-to-move-backups-between-storage-nodes?language=en_US
Alternatively, you can add a second location (e.g., a server disk) for the backups, but there is no other option.
