            Hi Patrick!
 
As I understand, you are describing a scenario, when a backup job is schedule but is missed due to the machine being offline (turned off or unavailable). You would like for the backup job to be re-tried once the machine is back online automatically.
 
This is option is enabled by default for all Backup Plans in Acronis Backup Cloud 7.5 (and earlier).
The trick, however, that this option cannot be turned off. We will add a separate option for that in Acronis Data Cloud 8.0 (this year) in Backup Plan settings.
 
Did I cover your question, or, perhaps, you've seen cases, when the above statements are not actual?
