Hello!
That message usually appears when the update did not finish cleanly or one of the Acronis services failed to start.
Please try the following steps:

Restart the machine where the agent/management server is installed.


Make sure all Acronis services are running (they should restart automatically after reboot).


If the issue persists, repair the installation using the same build installer.


Check if any local firewall/AV is blocking the connection after the update.

If none of this helps, please open a ticket with our support team so they can review the logs directly and identify the cause:https://support.acronis.com/
Best regards.
