            Hi Danny,
Branding works for all subgroups of the reseller group where you set the branding, but not for the reseller group itself. The reseller group will inherit the top group branding. So in your case in the reseller group you should create subgroups and they will inherit the new branding.
As for moving backup accounts to another group, this feature is not supported. The workaround is to delete the old account and create a new account in another group with the same login. In case the accout already has some machines, you can re-register them using the new account: https://kb.acronis.com/content/55244. Remember that backups are associated with the backup account and will be removed when the account is deleted.
