Creating and editing baseline templates

You need to create a custom template if you require multi-tenant management of baselines, but templates are useful in single -tenant environments as well, as reusable sets of baselines.

The process from creation to the actual use of a custom baseline template consists of several steps:

Create a baseline template.

On this step, you create a container for the baselines.

Populate the template with baselines.

On this step, you can add baselines from the library of baselines.

Configure the parameters of particular baselines in the template.

Deploy the baseline template.

To create a baseline template

Required roles: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.
Above the list of baseline templates, click Create template.

In the Create baseline template dialog, enter a name for the template, and then click Create.

The new empty baseline template appears in the list. Now you need to edit it to add baselines. See Adding baselines to templates.

To edit a baseline template

Required roles: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.

In the list of baseline templates, click the one that you want to edit.

The template details appear to the right.

Click the tabs to view or edit the properties that you need.

Details - template name and default template status.

Baselines - baselines in the template.

Customers - customers to which the template is deployed.
