Importing baselines to templates

In a baseline template, you can import baselines that are already managed by another template. If you want to add baselines that are not managed by another template, see Adding baselines to templates.

To import baselines to a template

Required roles: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.

In the list of baseline templates, click the one to which you want to add a baseline.

The template details appear to the right.

Click Import baselines.

In the Add baselines dialog, select the name of the template that contains the baselines that you want to import.

Select the baselines that you want to import and click Import.

You can search the list by baseline name or category.

In the Confirm template change dialog, review the changes that you applied and click Confirm.
