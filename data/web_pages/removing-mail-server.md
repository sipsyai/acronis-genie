# Removing a mail server

Managing the backup and recovery of workloads and files > Email archiving > Configuring email archiving > Removing a mail server
Removing a mail server

You can remove a mail server that is added to the Cyber Protect console.

To remove a mail server

Log in to the Cyber Protect console as administrator.
Go to Devices > Mail servers.
On the main screen, click the name of the mail server, and then click Delete.
To confirm your choice, click Delete.

As a result, the mail server is removed from the Cyber Protect console. The email archive will be kept, but it will no longer be updated.
