# Sorting alerts

Understanding your current level of protection > Monitoring > The Alerts dashboard > Sorting alerts
Sorting alerts

When using the table view, you can sort the alerts in descending or ascending order.

To sort the alerts

In the Cyber Protect console, go to Monitoring > Alerts.

[If you use simple view] Click the table view icon.

Click the name of the column in which you want to sort the alerts.

You cannot sort the alerts by clicking the Message column.

As a result, the alerts are sorted and an arrow is shown next to the name of the selected column.
