# Files forwarding method

Working with plans > SIEM forwarding plans > SIEM forwarding methods > Files forwarding method
Files forwarding method

This method can be implemented with either a Window or a Linux writer device.

The SIEM data is written as files to a directory. You can specify a directory or accept the default.

Default directories

Windows writer device: C:\.

Linux writer device: /var/log/siem.

Only administrators have access to these files.
