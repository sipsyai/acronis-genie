# Recovering mailboxes

Managing the backup and recovery of workloads and files > Protecting Microsoft 365 data > Locally installed Agent for Office 365 > Recovering mailboxes
Recovering mailboxes
Click Microsoft Office 365.

Select the mailbox to recover, and then click Recovery.

You can search mailboxes by name. Wildcards are not supported.

If the mailbox was deleted, select it on the Backup storage tab, and then click Show backups.

Select a recovery point. Note that recovery points are filtered by location.
Click Recover > Mailbox.

In Target mailbox, view, change, or specify the target mailbox.

By default, the original mailbox is selected. If this mailbox does not exist, you must specify the target mailbox.

Click Start recovery.
