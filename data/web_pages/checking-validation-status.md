# Checking the validation status

Working with plans > Off-host data protection plans > Validation > Checking the validation status
Checking the validation status

You can check the validation status of a backup on the Devices tab and on the Backup storage tab.

You can see the status for each validation method and download the screenshot when the screenshot validation method is used.

To check the validation status

Devices
Backup storage
In the Cyber Protect console, go to Devices > All devices.
Select a workload, and then click Recovery.
[If more than one backup location is available] Select the backup location.

Select the backup for which you want to check the validation status.
