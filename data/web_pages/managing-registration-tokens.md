# Managing registration tokens

Installing and deploying Cyber Protection agents > Registration of workloads > Registering workloads by using the graphical user interface > Managing registration tokens
Managing registration tokens

You can view and delete the active registration tokens.

To view the registration tokens
To delete a registration token
Log in to the Cyber Protect console.
Click Devices > All devices > Add.

Scroll down to Registration token, and then click Manage active tokens.

A list with the active tokens that are generated for the tenant opens on the right.

For security reasons, in the Token column, only the first two characters of the token value are shown.
