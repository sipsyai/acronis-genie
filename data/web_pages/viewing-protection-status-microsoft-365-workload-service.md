Viewing the protection status of a Microsoft 365 workload per service

Depending on the type of Microsoft 365 workload, you can protect the following services:

Workload	Services
Groups*	Mailbox
Site
Public folders	Mailbox
Site collections	Site
Teams	Teams Channels
Users	Mailbox
OneDrive

* Groups contain modern (team) sites and group mailboxes. Groups that are part of a team might be double backed-up if they are protected both as a group and as part of the team. For more information, see Avoiding duplicate backups of Microsoft 365 groups.

If more than one service is available for a workload but only one is protected, the workload is partially protected. For example, if the mailbox of a user is backed-up but the OneDrive files are not, this user is partially protected.

To check the protection status per service

Log in to the Cyber Protect console.
Go to Devices > Microsoft 365 > Microsoft 365 organization.
Select a workload, and then click Details.

Check the Services section.
