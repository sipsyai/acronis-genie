# Changing the access mode

Additional Cyber Protection tools > Archival storage > Managing buckets in archival storage > Changing the access mode
Changing the access mode

You can change the access mode for a bucket. For more information, see Access mode.

To change the access mode

In the Cyber Protect console, go to Backup storage > Archival storage.
Click the bucket that you want to edit.
On the Bucket tab, select the access mode.
Click Save.
