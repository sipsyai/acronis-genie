Reviewing baseline templates

Using the Microsoft 365 security monitoring and management service, you can review the details of all current tenant baseline templates, including the tenants attached to each template. You can also enable auto remediation for selected baselines.

To review baseline templates

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.

Click a template in the list to open its details.

On the Details tab, you can edit the template name, and check if the template is set as default.

On the Baselines tab, you can view details about the baselines in the template, including whether a baseline requires user input, and any associated Microsoft service plans.

User input indicates that some values for the baseline must be defined by the user before applying the template to the tenant.

Some baselines also require specific Microsoft service plans to be applied, such as AAD_PREMIUM, and AAD_PREMIUM_P2. If a tenant does not have these plans, the baseline is omitted for that tenant.

To enable auto remediation for a baseline, select the baseline in the list, and then click Enable auto remediation.

On the Customers tab, view the customers to which a baseline template is applied and revoke the template from specific customers.
