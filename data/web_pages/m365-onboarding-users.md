# Onboarding users

RMM > Managing and monitoring the Microsoft 365 environment > Managing Microsoft 365 users > Onboarding users
Onboarding users

You can onboard new users to a Microsoft 365 tenant through the Microsoft 365 management service.

To onboard a new user

In the Cyber Protect console, go to Microsoft 365 management > Users.

Click Onboard user.

In the dialog, define the following user details:

First name: The user's first name.

Last name: The user's last name.

Display name: The name shown to email recipients.

User name: The user name when signing in.

Domain: Select the relevant Microsoft 365 domain from the dropdown.

Location: Select the relevant country from the dropdown.

Password: Enter the password required to sign in.

In the Contact information section, enter the relevant information.

Select the Prompt the user to change their password when they first sign in checkbox if you want users to change their passwords when first signing in.

Click Create.
