Configuring proxy server settings in Cyber Protect Monitor

You can configure the proxy server settings in Cyber Protect Monitor. The configuration will affect all agents that are installed on the machine.

To configure the proxy server settings

Open Cyber Protect Monitor, and then click the gear icon in the top right corner.
Click Settings, and then click Proxy.
Enable the Use a proxy server switch, and then enter the proxy server address and port.

[If the proxy server access is password-protected] Enable the Password required switch, and then enter the user name and password to access the proxy server.

The password can include lowercase and uppercase alphanumeric characters, and the following special characters:

! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` } | { ~

Click Save.

The proxy server settings are saved in the http-proxy.yaml file.
