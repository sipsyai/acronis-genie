# Creating a static group

Managing workloads in the Cyber Protect console > Device groups > Creating a static group
Creating a static group

You can create an empty static group and add workloads to it.

Alternatively, you can select workloads and create a new static group from your selection.

You cannot create device groups within any All-type group, such as the root group All devices, or built-in groups like Machines with agents > All, Microsoft 365 > your organization > Users > All users.

To create a static group

In the main window
In the group tree
From selection

Click Devices, and then select the root group that contains the workloads for which you want to create a static group.

To create a nested group, navigate to an existing static group.

Creating nested static groups is not available for cloud-to-cloud workloads.

Click + New static group below the group tree or click New static group in the Actions pane.
Specify a name for the new group.

Add a comment for the group.

Click OK.
