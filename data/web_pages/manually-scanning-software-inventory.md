Starting a software inventory scan manually

You can start a software inventory scan manually from the Software inventory screen, or from the Software tab on the Inventory screen.

Prerequisites

The device uses Windows or macOS operating system.
The device has the required license.
From Software inventory
From Software

To start a software inventory scan from the Software inventory screen

In the Cyber Protect console, go to Software management.

Click Software inventory.

In the Group by: dropdown menu, select Devices.

Find the device which you want to scan, and then click Scan now.
