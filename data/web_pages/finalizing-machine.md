Finalizing a virtual machine running from a backup

You can make a virtual machine that is running from a backup permanent, by finalizing it.

To finalize a virtual machine that is running from a backup (Instant Restore)

VMware ESXi
Microsoft Hyper-V
Proxmox VE
In the Cyber Protect console, select a machine that is running from a backup.
In Actions, click Finalize.
Specify a name for the permanent virtual machine.
Select the disk provisioning mode.
Click Finalize.
