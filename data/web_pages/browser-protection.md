# Browser protection

Configuring your antivirus and antimalware protection > URL filtering > URL filtering settings > Browser protection
Browser protection

Enable Browser protection to install the URL filtering extension to your browser.

The URL filtering extension enhances URL filtering by supporting HTTPS websites that use modern protocols such as HTTP/2 and QUIC. It ensures that web filtering stays accurate and reliable even with newer, faster types of internet traffic.

The URL filtering extension applies the same rules defined in your URL filtering settings for accessing malicious websites. See Malicious website access.

Install the URL filtering extension

URL filtering extension is available only for Google Chrome, version 130 or later.

Turn on the Browser protection switch.

Wait for the alert Browser protection is almost ready to appear.

Restart your browser to complete the installation.

In the URL filtering extension pop-up window, turn on the Log collection switch.

This will allow the collection of logs to the browser storage.

The default browser storage capacity is 50 MB. When the browser storage is full, the oldest logs will be removed

The URL filtering extension is installed and you can collect logs to monitor its performance.
