Viewing the software inventory of a single device

You can view a list of all the software applications that are installed on a single device, and detailed information about the applications, such as status, version, vendor, installation date, last run, and location.

Prerequisites

The device uses Windows or macOS operating system.

The device has the required license.

At least one software inventory scan on the device finished successfully.

From Software inventory
From Devices

To view the software inventory of a single device from the Software inventory screen

In the Cyber Protect console, go to Software management.

Click Software inventory.

In the Group by: dropdown menu, select Devices.

Find the device that you want to inspect, by using one of the following options.

Use Filter:

Click Filter.

In the Device name field, select the name of the device that you want to view.

Click Apply.

Use Search:

Click Search.

Type the full device name or part of the device name.
