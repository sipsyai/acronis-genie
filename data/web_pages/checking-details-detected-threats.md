Checking the details about a detected threat

You can understand more about detected threats by checking the backup archive, the infected email, or the Alerts tab in the Cyber Protect console.

To check the details for a detected threat

In the backup archive
In a backed-up email
In Monitoring > Alerts
In the Cyber Protect console, go to Backup storage > Backups, and then select Cloud application storage.
Select the backup archive that you want to check, and then click Show backups.
Select a backup that is marked with a red icon for an infected file or malicious URL.
For more information about the threat, click the Infected files or Malicious URLs link.
