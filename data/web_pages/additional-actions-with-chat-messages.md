Additional actions with chat messages

In active chat sessions, you can edit or delete messages that you sent to the remote user.

Edit message
Delete message

You can edit any message that you sent in the chat session.

To edit a message in the chat session

In the chat window, locate the message that you want to edit.
Hover over the message, and click the pencil icon that appears.

Edit the message, and then press Enter.

The changes are saved. The message is tagged as Edited.
