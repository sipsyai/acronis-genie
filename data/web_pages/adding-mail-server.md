# Adding a mail server

Managing the backup and recovery of workloads and files > Email archiving > Configuring email archiving > Adding a mail server
Adding a mail server

Prerequisites

The following quotas must be enabled for the tenant in Management Portal:

Microsoft 365 email archiving seats
Archiving storage
Both quotas are required. If any of them is removed or misconfigured after the initial configuration, email archiving will be paused for all plans of the tenant until the quota is restored.

To add a mail server

Log in to the Cyber Protect console as administrator.

Go to Devices, and then click Add > Microsoft 365 Business – Email archiving.

You are redirected to the Microsoft 365 login page.

On the Microsoft login page, sign in as global administrator.

Review the list of required permissions, and then click Accept.

The Microsoft 365 mail server appears in the Cyber Protect console, on the Devices > Mail servers tab.

On the screen that opens, configure the archiving plan as described in Creating an archiving plan.
