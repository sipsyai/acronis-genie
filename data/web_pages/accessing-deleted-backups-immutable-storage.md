Accessing deleted backups in immutable storage

During the retention period, you can access deleted backups in immutable storage and recover data from them.

A deleted backup can be part of a deleted archive or non-deleted archive.

To allow access to deleted backups, port 40440 on the backup storage should be enabled for incoming connections.

To access a deleted backup

In a deleted archive
In a non-deleted archive
On the Backup storage tab, select the cloud storage that contains the deleted backup.
To see the deleted archives, click Show deleted.
Select the archive that contains the backup that you want to recover.
Click Show backups.
Select the backup that you want to recover.
Proceed with the recovery operation, as described in Recovery.
