# Importing a protection plan

Working with plans > Protection plans and modules > Actions with protection plans > Importing a protection plan
Importing a protection plan

You can import protection plans that were exported in a JSON file. Thus, you can save time and ensure consistency of the settings across tenants.

This feature is not available for plans at the partner level. For more information about this level, see Administration levels in the Cyber Protect console.

Prerequisites

A JSON file with the plan configuration is available on the machine from which you are logged in to the console.

To import a protection plan

In the Cyber Protect console, go to Management > Protection plans.
In the Actions menu, click Import.
In the window that opens, browse to the JSON file.

Click the file, and then click Open.

The protection plan appears on the screen. You can now apply it to workloads.
