# Recovering system state

Managing the backup and recovery of workloads and files > Recovery > Recovering system state
Recovering system state

In the Cyber Protect console, you cannot recover backups in tenants in Compliance mode. See Recovering backups in tenants in Compliance mode.

Select the machine for which you want to recover the system state.
Click Recovery.
Select a system state recovery point. Note that recovery points are filtered by location.
Click Recover system state.
Confirm that you want to overwrite the system state with its backed-up version.

You can check the recovery progress in the Cyber Protect console, on the Activities tab.
