Unregistering workloads

From the command-line interface, you can unregister a protection agent without uninstalling it.

To unregister a workload

In Windows
In Linux
In macOS
Virtual appliance

In Command Prompt, run the following command:

"%ProgramFiles%\BackupClient\RegisterAgentTool\register_agent.exe" -o unregister

For example:

"C:\Program Files\BackupClient\RegisterAgentTool\register_agent.exe" -o unregister
