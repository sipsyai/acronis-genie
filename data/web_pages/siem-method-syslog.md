# Syslog server forwarding method

Working with plans > SIEM forwarding plans > SIEM forwarding methods > Syslog server forwarding method
Syslog server forwarding method

A syslog server is a centralized system that receives and stores log messages using the Syslog protocol.

Only Linux devices can be used for this method.

To review forwarded SIEM data

Execute: cat /var/log/syslog

Execute: cat /var/log/messages
