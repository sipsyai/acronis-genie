# Creating an archiving plan

Managing the backup and recovery of workloads and files > Email archiving > Archiving plans > Creating an archiving plan
Creating an archiving plan

You can create an archiving plan when you add a new mail server to the Cyber Protect console, or create an archiving plan for an existing mail server on the Cyber Protect console.

New mail server
Existing mail server

To create an archiving plan

Add the mail server to the Cyber Protect console, as described in Adding a mail server.

An archiving plan opens.

To change the archiving plan name, click the pencil icon, specify a new name, and then click OK.

In Import existing emails, select All mailboxes, and then click Done.

Click Encryption, specify and confirm the encryption password, and then click Save.

There is no way to browse the email archive and recover data from it if you lose or forget this password.

Click Create.

As a result, an email archiving plan is created and applied to the mail server, and the email archive is created.

The initial import will take some time, depending on the selected import scope and period. All new emails will be continuously added to the archive, as they are sent or received.
