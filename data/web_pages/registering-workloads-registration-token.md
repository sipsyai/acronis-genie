Registering workloads by using a registration token

You can modify the default installation procedure and select registration with a registration token, instead of registration in the Cyber Protect console.

To register a workload by using a registration token

In the installation wizard, click Customize installation settings.
In the Registration settings section, click Change.
Select Use registration token.
Enter the registration token.
Click Done, and then complete the installation.
