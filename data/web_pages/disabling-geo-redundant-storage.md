# Disabling Geo-redundant storage

Additional Cyber Protection tools > Geo-redundant storage > Disabling Geo-redundant storage
Disabling Geo-redundant storage

You can disable Geo-redundant storage in the Cyber Protect console.

To disable Geo-redundant storage

Log in to the Cyber Protect console as administrator.
Go to Settings > System settings.
Collapse the list of default backup options, and then click Geo-redundant cloud storage.
Disable the Geo-redundant cloud storage switch.
Click Save.
To confirm your choice, type Disable, and then click Disable.

As a result, Geo-redundant storage is disabled. Replicated data will be deleted within one day.
