# Configuring action on detection for data loss prevention

Configuring your GenAI Protection > GenAI Protection settings > Data loss prevention > Configuring action on detection for data loss prevention
Configuring action on detection for data loss prevention

To set the action on detection for data loss prevention

In the GenAI Protection module of a protection plan, click Data loss prevention.

In the dialog, specify the Action on detection

Detect only - detects sensitive data that is sent to AI applications, and records details in the events log.

Block - prevents sending sensitive data to AI applications, records details in the events log, and notifies the user.

Click Done.
