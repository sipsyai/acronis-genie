Working with plans > Understanding plans > Default plans > Setting plans as default
Setting plans as default

You can set one plan of the supported plan types (protection plan, monitoring plan, or remote management plan) as default.

The feature is not available for users who are assigned the Read-only administrator role for the Protection service.

Protection plans
Remote management plans
Monitoring plans

Prerequisites

At least one protection plan is created for your tenant. For more information, see Creating a protection plan.

To set a protection plan as default

On the Protection plans screen, select the plan that you want to set as default, and then click Details.
Click the ellipsis icon, and then click Set as default.

In the confirmation window, click Set.

On the Protection plans screen, the Default tag appears next to the plan name.
