# Deleting software packages

RMM > Managing software > Working with software repositories and software packages > Deleting software packages
Deleting software packages

The availability of this feature depends on the license that you have.

You can delete software packages that you do not need anymore.

Prerequisites

You have one of the following roles for Cyber Protection: Company administrator or Cyber administrator.

To delete a software package

In the Cyber Protect console, go to Software management > My packages.
For the package that you want to edit, click the ... icon, and then click Delete.
In the confirmation window, click Delete.

If the package was uploaded manually, it will be permanently deleted. If the package was added from the library, it will be deleted only from your packages.
