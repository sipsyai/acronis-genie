Managing the backup and recovery of workloads and files > Protecting Microsoft 365 data > Locally installed Agent for Office 365 > Changing the Microsoft 365 access credentials
Changing the Microsoft 365 access credentials

You can change access credentials for Microsoft 365 without re-installing the agent.

To change the Microsoft 365 access credentials

Click Devices > Microsoft Office 365.

Select the Microsoft 365 organization.

Click Specify credentials.

Enter your application ID, application secret, and Microsoft 365 tenant ID. For more information on how to find these, refer to Obtaining application ID and application secret.

Click OK.
