Editing baselines in templates

You can edit the baselines that are managed by custom templates to change their parameters as needed.

To edit a baseline in a template

Required roles: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.

In the list of baseline templates, click the one that contains the baseline to edit.

The template details appear to the right, and the list of baselines is displayed on the Baselines tab.

Click the context menu at the end of the baseline's row and select Edit.

The Edit baseline dialog opens.

Apply the changes that you need and click Save in the upper right.
