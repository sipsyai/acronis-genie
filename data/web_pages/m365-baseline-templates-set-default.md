RMM > Managing and monitoring the Microsoft 365 environment > Managing tenant baseline templates > Setting a default baseline template
Setting a default baseline template

The default template is used when onboarding new tenants. Initially, the built-in template Best practices is set as default. As Partner administrator, you can change the default template as needed, or remove the default flag from a template.

Only one template can be set as default for all customers under the Microsoft 365 security posture management service.

To set a default baseline template

Required role: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.
In the list of baseline templates, select the check box of the one that you want to make the default.
Click Set as default above the list.

The Default template column indicates the default template with Yes.

To remove the default flag from a baseline template

Required role: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.
In the list of baseline templates, select the check box of the default baseline.
Click Clear default above the list.
