Access mode

In archival storage, you can control data access and security by configuring buckets as public or private.

Public mode makes a bucket and its contents available to anyone who has the correct URL.

You can use this mode to share non-sensitive files.

Private mode restricts bucket access to authorized users.

You can use this mode for sensitive or business-critical data.
