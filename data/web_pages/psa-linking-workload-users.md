# Linking workloads to specific users

Managing workloads in the Cyber Protect console > Linking workloads to specific users
Linking workloads to specific users
This feature is only available if the PSA service is enabled.

By linking a workload to a specific user, you can automatically link the workload to new service desk tickets created by or assigned to the user.

To link a workload to a user

Go to Devices > All devices, and then select the relevant workload.
In the Actions pane, select Link to a user.

Select the relevant user.

You can also change the selected user for existing linked workloads, as required.

Click Done.

The selected user is now displayed in the Linked user column.

To unlink a workload from a user

Go to Devices > All devices, and then select the relevant workload.
In the Actions pane, select Link to a user.
Click Unlink user.
Click Done.
