# Search queries

Managing the backup and recovery of workloads and files > Email archiving > Search queries
Search queries

You can find specific emails in the email archive by using search queries, and then save the queries for future use.

Search queries form the basis for creating retention rules and legal hold rules.

You can link a saved search query to a specific email archive. Such a search query can run only on the linked email archive.

If you do not link the saved search query to an email archive, it can run on any email archive. You can use non-linked search queries in retention rules and legal hold rules that are applied to multiple email archives.
