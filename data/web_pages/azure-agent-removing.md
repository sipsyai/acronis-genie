Installing and deploying Cyber Protection agents > Deploying virtual appliances > Deploying Agent for Azure > Removing the Agent for Azure virtual appliance
Removing the Agent for Azure virtual appliance

The Agent for Azure virtual appliance is automatically removed when you remove the corresponding Microsoft Azure subscription.

To remove the Agent for Azure virtual appliance

In the Cyber Protect console, go to Devices > Microsoft Azure.

Click the Subscriptions node in the hierarchical tree, and then select the relevant subscription in the displayed list to the right.

In the right pane, click Delete.

In the displayed confirmation dialog, click Remove.

When the subscription is removed, the Agent for Azure virtual appliance is de-provisioned and does not use Microsoft Azure resources.

You cannot remove a subscription if it is currently in use by Microsoft Azure virtual machines.
