Viewing vulnerabilities at the partner level

On the Vulnerabilities screen, you can view a list of all the vulnerabilities that were found on your customers' devices, if vulnerability assessment was performed at least once and some vulnerabilities were found.

To view details about a vulnerability

Log in to the Cyber Protect console at the partner level (All customers), and then go to Software management > Vulnerabilities.

The list of vulnerabilities shows both vulnerabilities that have patches to be installed, and those that do not have suggested patches. You can use the filter to show only vulnerabilities with patches.

Field	Description
Name	Name of the vulnerability
Severity

Severity of found vulnerability. The following levels can be assigned according to the Common Vulnerability Scoring System (CVSS):

Critical: 9 - 10 CVSS
High: 7 - 9 CVSS
Medium: 3 - 7 CVSS
Low: 0 - 3 CVSS
None

Affected products	Name of the product that is affected by the vulnerability


Customers

Number of customers that have devices on which the vulnerability was found. If you click the number, the Affected tenants tab opens, where you can see a list of the affected customers.
Devices	Number of devices on which the vulnerability was found. If you click the number, the Affected devices tab opens, where you can see more information about the affected devices, such as name, customer account under which the device is registered, and MAC address.
Patches	Number of patches available to resolve the vulnerability. The value that is displayed is a link to the Patches screen, where results are filtered to show only the information that is relevant to this specific patch.
Published	Date and time when the vulnerability was published in Common Vulnerabilities and Exposures (CVE)
Detected	First date when an existing vulnerability was detected on devices

Click a vulnerability from the list.

An additional pane opens. You can see more details that are related to the vulnerability on three tabs: Details & Patches, Affected tenants, and Affected devices.
