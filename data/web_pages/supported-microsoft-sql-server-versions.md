# Supported Microsoft SQL Server versions

Managing the backup and recovery of workloads and files > Supported operating systems and environments for backup and recovery > Supported Microsoft SQL Server versions
Supported Microsoft SQL Server versions
Microsoft SQL Server 2022
Microsoft SQL Server 2019
Microsoft SQL Server 2017
Microsoft SQL Server 2016
Microsoft SQL Server 2014
Microsoft SQL Server 2012
Microsoft SQL Server 2008 R2
Microsoft SQL Server 2008
Microsoft SQL Server 2005

The SQL Server Express editions of the above SQL server versions are supported as well.

Microsoft SQL backup is supported only for databases running on NFTS, REFS, and FAT32 file systems. ExFat is not supported.
