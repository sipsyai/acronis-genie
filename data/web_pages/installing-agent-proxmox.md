Installing Agent for Proxmox

Agent for Proxmox is bundled with Agent for Linux (64-bit), and requires that Agent for Linux is also installed on the protected host.

You must install Agent for Proxmox locally, on the protected host, by using one of the following options:

Graphical user interface

For more information, see Installing protection agents in Linux.

Command-line interface

For more information, see Installing and uninstalling protection agents in Linux.

After you install the agent on the host, you can see all virtual machines and containers that are available on that host in the Cyber Protect console, on the Devices > Proxmox tab.
