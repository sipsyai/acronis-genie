# Configuring action on detection for prompt injection protection

Configuring your GenAI Protection > GenAI Protection settings > Prompt injection protection > Configuring action on detection for prompt injection protection
Configuring action on detection for prompt injection protection

To set action on detection for prompt injection protection

In the GenAI Protection module of a protection plan, click Prompt injection protection.

Specify the Action on detection:

Detect only - detects potential prompt injection attacks, and records details in the events log.

Block - blocks potential prompt injection attacks, records details in the events log, and notifies the user.

Click Done.
