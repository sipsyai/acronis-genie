# Exporting baselines to a CSV file

RMM > Managing and monitoring the Microsoft 365 environment > Managing tenant baseline templates > Exporting baselines to a CSV file
Exporting baselines to a CSV file

You can export baselines to a CSV file for reporting purposes.

To export baselines to a CSV file

Required roles: Partner administrator; Customer administrator

In the Cyber Protect console, go to Microsoft 365 management > Baselines.

In the baselines list, select the checkboxes of the baselines that you want to export, and then click Export CSV above the list.

In the Export baselines dialog, select whether to export all fields or only the fields with deviating values, and then click Export.

The CSV file downloads to the default download location configured in your browser.
