# Clearing default plans

Working with plans > Understanding plans > Default plans > Clearing default plans
Clearing default plans

You can clear the default protection plan. After you do that, the plan will no longer be preselected in the plan selection fields.

The feature is not available for users who are assigned the Read-only administrator role for the Protection service.

Protection plans
Remote management plans
Monitoring plans

Prerequisites

A protection plan is created for your tenant and set as default. For more information, see Creating a protection plan.

To clear a default protection plan

On the Protection plans screen, select the default plan, and then click Details.
Click the ellipsis icon (...), and then click Clear default.

In the confirmation window, click Clear.

On the Protection plans screen, the Default tag that is next to the plan name disappears.
