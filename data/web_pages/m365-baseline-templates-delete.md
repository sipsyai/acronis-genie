# Deleting baseline templates

RMM > Managing and monitoring the Microsoft 365 environment > Managing tenant baseline templates > Deleting baseline templates
Deleting baseline templates

You can delete baseline templates if you no longer need them. Because the templates are just containers for baselines, the baselines will not be affected and you can continue using them as needed.

To delete a baseline template

Required roles: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baseline templates.

In the list of baseline templates, select the check box of the template that you want to delete.

Click Delete template above the list.

[If the template is deployed to customers] In the Delete template dialog, review the list of affected customers and decide how to handle the baselines that are managed by the template:

Keep baselines as unmanaged

If you select this option, the baselines will remain assigned to the customers but will no longer be managed by the template.

Remove baselines from the above customers

The baselines will be removed from the customers, but will not be deleted, so you can use them later as needed.

Click Confirm.
