# Changing the SQL Server or Exchange Server access credentials

Managing the backup and recovery of workloads and files > Protecting Microsoft applications > Changing the SQL Server or Exchange Server access credentials
Changing the SQL Server or Exchange Server access credentials

You can change access credentials for SQL Server or Exchange Server without re-installing the agent.

To change the SQL Server or Exchange Server access credentials

Click Devices, and then click Microsoft SQL or Microsoft Exchange.
Select the Always On Availability Group, Database Availability Group, SQL Server instance, or Exchange Server for which you want to change the access credentials.
Click Specify credentials.
Specify the new access credentials, and then click OK.

To change the Exchange Server access credentials for mailbox backup

Click Devices > Microsoft Exchange, and then expand Mailboxes.
Select the Exchange Server for which you want to change the access credentials.
Click Settings.
Under Exchange administrator account, specify the new access credentials, and then click Save.
