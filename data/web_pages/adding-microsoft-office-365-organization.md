# Adding a Microsoft 365 organization

Managing the backup and recovery of workloads and files > Protecting Microsoft 365 data > Adding a Microsoft 365 organization
Adding a Microsoft 365 organization
This topic applies to Microsoft 365 Business – Backup and Direct Backup to Microsoft Backup Storage.

With Microsoft 365 Business – Backup, you can add a Microsoft 365 organization to a customer tenant or unit.

With Direct Backup to Microsoft 365 Backup Storage, you can add a Microsoft 365 organization to a customer tenant.

To add a Microsoft 365 organization, you must have administrator privileges for the customer tenant or the unit.

You can add multiple Microsoft 365 organizations in the tenant, but you cannot add the same Microsoft 365 organization in both Microsoft 365 Business – Backup and Direct Backup to Microsoft 365 Backup Storage.

To add a Microsoft 365 organization

[To add a Microsoft 365 organization in a customer tenant] Log in to the Cyber Protect console as an administrator.

[To add a Microsoft 365 organization in a unit, only for Microsoft 365 Business – Backup] Depending on your role, do the following:

[For unit administrators] Log in to the Cyber Protect console as a unit administrator.

[For company administrators acting at the unit level] Navigate to the unit in which you want to add the organization.

Click Devices > Add.

Select the backup solution.

Depending on the services that are enabled for your tenant, you can choose between the following options:

Microsoft 365 Business – Backup

Direct Backup to Microsoft 365 Backup Storage

Microsoft 365

This option is shown when both Microsoft 365 Business – Backup and Direct Backup to Microsoft 365 Backup Storage are enabled for your tenant. For more information about the differences between these solutions, see Comparison of backup solutions for Microsoft 365 data.

[If you selected Microsoft 365] Select Microsoft 365 Business – Backup or Direct Backup to Microsoft 365 Backup Storage, and then click Continue.

On the Microsoft 365 login page, sign in with a Microsoft 365 global administrator credentials.

This account is used only to add the Microsoft 365 organization to the Cyber Protect console. For security reasons, you can use a dedicated global administrator account for this operation. After you add the organization, you can delete the dedicated account, which is no longer required.

Microsoft 365 displays a list of permissions that are necessary to back up and recover your organization's data.

Confirm that you grant the required permissions.

Depending on the selected solution, you will see the Microsoft 365 organization in the Cyber Protect console, on the Devices > Microsoft 365 tab or on the Devices > Direct Backup for Microsoft 365 tab.
