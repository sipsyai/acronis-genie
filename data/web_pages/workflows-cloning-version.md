# Cloning a workflow version

Managing and assigning automation workflows > Workflow versions > Cloning a workflow version
Cloning a workflow version

You can clone any existing workflow version that is currently set as Enabled.

When logged in as the Company administrator, Administrator, or Cyber administrator, you can clone any workflow version, including the versions of workflows in Draft status.

To clone a workflow version

In the Cyber Protect console, go to Management > Workflows.

In the main Workflows screen, select the relevant workflow, and then click the Versions tab.

In the relevant version row, click the ellipsis (...) icon, and then select Clone.

The cloned version is automatically set as Draft and is also disabled (even if you cloned a workflow set as Enabled).
