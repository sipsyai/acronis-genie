Recovering emails, folders, and mailboxes

You can recover emails, folders, and mailboxes to the original or a non-original mailbox.

Folders
Mailboxes

To recover emails

Log in to the Cyber Protect console as administrator.
Go to Backup storage > Cloud application backups, and then select the email archive from which you want to recover data.
Click Browse archive.
[If a privacy warning is shown] In the Privacy warning dialog, click Proceed.
In the Encryption password dialog, specify the password, and then click Proceed.

Search for the emails that you want to recover.

You can use ad-hoc search queries and saved search queries. For more information, see Searching for emails.

Select the emails that you want to recover, and then click Recover emails.

[If multiple mail servers are added to the Cyber Protect console] Select the organization in which you want to recover data.

In Organization, click Select.

Select the organization, and then click Select.

In Recovery location, select the mailbox in which you want to recover the emails.

[If you selected Another mailbox ] Select the target mailbox.

In Recover to, click Select.

Select the target mailbox, and then click Select.

Click Start recovery.

As a result, the incoming emails from the selected mailbox are recovered to the Inbox folder of the target mailbox, and the outgoing emails from the selected mailbox are recovered to the Sent folder of the target mailbox.
