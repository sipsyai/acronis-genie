# Recovering Proxmox workloads

Managing the backup and recovery of workloads and files > Protecting Proxmox virtual machines and containers > Recovering Proxmox workloads
Recovering Proxmox workloads

You can recover a backed-up Proxmox virtual machine as:

A physical machine.
The original or a new virtual machine on the same hypervisor.
A new machine on a different hypervisor. See Cross-platform recovery.

For more information about recovering a virtual machine, see Recovering a virtual machine as a physical machine and Recovering a virtual machine as a virtual machine.

You can recover a backed-up Proxmox container as:

The original container.
A new Proxmox container. Cross-platform recovery is not supported.

For more information about recovering a container, see Recovering containers.

You can recover files from backed-up Proxmox virtual machines and containers to the Proxmox host or network storage attached to it. You cannot perform file-level recovery to a virtual machine or container.
