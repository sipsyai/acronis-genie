# Importing scripting plans

Managing workloads in the Cyber Protect console > Cyber Scripting > Scripting plans > Importing scripting plans
Importing scripting plans

You can import scripting plans that were exported in a JSON file. Thus, you can save time and ensure consistency of the settings across tenants.

Prerequisites

A JSON file with the plan configuration is available on the machine from which you are logged in to the console.

To import a scripting plan

In the Cyber Protect console, go to Management > Scripting plans.
Click Import.
In the window that opens, browse to the JSON file.

Click the file, and then click Open.

The scripting plan appears on the screen. You can now apply it to workloads.
