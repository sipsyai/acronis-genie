# Settings of the Last system restart monitor

RMM > Monitoring the health and performance of workloads > Configurable monitors > Settings of the Last system restart monitor
Settings of the Last system restart monitor

Last system restart when the workload was last restarted.

You can configure the following setting for the monitor.

Setting	Description
The workload has not been restarted for

The period (number of days) since the last restart of the workload. If the workload has not been restarted for a longer period than the period you specify, the system will generate an alert.

Enter an integer value in the range 1-180 (days). The default value is 30.
