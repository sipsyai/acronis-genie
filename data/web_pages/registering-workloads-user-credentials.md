Registering workloads by using user credentials

You can modify the default installation procedure and select registration with a user name and password, instead of registration in the Cyber Protect console.

To register a workload by using a user name and password

In the installation wizard, click Customize installation settings.
In the Registration settings section, click Change.
Select Use credentials.

Specify the user name and password of the account under which you want to register the workload.

This account must be an account in a customer tenant.

You can use only accounts for which two-factor authentication is not enabled.

Click Done, and then complete the installation.
