# Resetting user passwords

RMM > Managing and monitoring the Microsoft 365 environment > Managing Microsoft 365 users > Resetting user passwords
Resetting user passwords

You can reset a specific user's Microsoft 365 password.

To reset a user's Microsoft 365 password

In the main Users screen, click the relevant user row.

Click Reset password.

In the displayed dialog, enter the user's new password.

If you want to force the user to change their password, select Require this user to change their password when they first sign in.

Click Save.

In the confirmation dialog, click Close.
