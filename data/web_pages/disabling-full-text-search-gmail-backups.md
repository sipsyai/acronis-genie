# Disabling full-text search for Gmail backups

Managing the backup and recovery of workloads and files > Search in cloud-to-cloud backups > Disabling full-text search for Gmail backups
Disabling full-text search for Gmail backups

You might want to disable full-text search if you need to keep the size of the search index minimal.

If you enable full-text search again later, all archives that are created by this backup plan will be indexed again. This is a time-consuming operation.

To disable full-text search

When creating or editing a backup plan, click the gear icon in the upper right corner.
On the Full-text search tab, disable the switch.
Click Done.
[When creating a plan] Click Apply.
[When editing a plan] Click Save settings.
