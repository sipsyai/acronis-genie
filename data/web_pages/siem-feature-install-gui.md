Installing the SIEM feature using GUI

You can install the SIEM feature on your SIEM writer device using the Acronis agent installation GUI.

You must explicitly request the SIEM Log Forwarder feature during the installation.

Prerequisite

Read the instructions found in Installing protection agents by using the graphical user interface.

To install the SIEM feature using GUI

Run the installation file.

The initial installation screen appears.

Click Customize installation settings.

The Installation settings screen appears.

Click Change to specify What to install.

The What to install screen appears.

Ensure that you select the SIEM Log Forwarder check-box.

Click Done.

Make other required changes to the installation settings.

For more information, see Installing protection agents by using the graphical user interface.

Click Install to install the Acronis agent with the SIEM log forwarding feature enabled.
