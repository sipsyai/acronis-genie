# File filters (Inclusions/Exclusions)

Managing the backup and recovery of workloads and files > Backup options > File filters (Inclusions/Exclusions)
File filters (Inclusions/Exclusions)

You can use file filters to include only specific files and folders in a backup, or to exclude specific files and folders from a backup.

File filters are available for entire machine backups, disk-level backups, and file-level backups, unless stated otherwise.

File filters are not available with the XFS, JFS, exFAT, and ReiserFS4 file systems. For more information, see Supported file systems.

File filters are not applicable to dynamic disks (LVM or LDM volumes) of virtual machines that are backed up in the agentless mode, for example, by Agent for VMware, Agent for Hyper-V, or Agent for Scale Computing.
