# Deleting a workflow version

Managing and assigning automation workflows > Workflow versions > Deleting a workflow version
Deleting a workflow version

As the Company administrator, Administrator, or Cyber administrator, you can delete workflow versions of workflows that you created or that were assigned to you.

You can delete only workflow versions in Draft state.

To delete a workflow version

In the Cyber Protect console, go to Management > Workflows.

In the main Workflows screen, select the relevant workflow, and then click the Versions tab.

In the relevant version row, click the ellipsis (...) icon, and then select Delete.

In the confirmation dialog, click Delete.
