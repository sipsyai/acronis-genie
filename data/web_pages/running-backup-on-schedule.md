# Running a backup on a schedule

Managing the backup and recovery of workloads and files > Backup schedule > Running a backup on a schedule
Running a backup on a schedule

To run a backup automatically at a specific time or on a specific event, enable a schedule in the protection plan.

To enable a schedule

In the protection plan, expand the Backup module.
Click Schedule.
Enable the schedule switch.
Select the backup scheme.

Configure the schedule as required, and then click Done.

For more information about the available scheduling options, see Schedule by time and Schedule by events.

Configure start conditions or additional scheduling options.

Save the protection plan.

As a result, a backup operation starts every time when the schedule conditions are met.

To disable a schedule

In the protection plan, expand the Backup module.
Click Schedule.
Disable the schedule switch.
Save the protection plan.

As a result, the backup runs only if you start it manually.

If the schedule is disabled, the retention rules are not applied automatically. To apply them, run the backup manually.
