Additional actions with the remote command-line console

After you remotely access the command line of a managed workload, you can perform the following actions from the console: copy content from the console, paste content into the console, clear the console, and export the content of the console.

Copy
Paste
Clear console
Export

You can copy parts of the content from the console.

To copy from the console

In the command-line console, select the text that you want to copy.

Click Copy.

The selected text is copied.
