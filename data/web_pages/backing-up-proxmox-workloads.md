Managing the backup and recovery of workloads and files > Protecting Proxmox virtual machines and containers > Backing up Proxmox workloads
Backing up Proxmox workloads

You can back up Proxmox virtual machines and containers.

To back up a workload, create a protection plan. For more information, see Creating a protection plan.

In the protection plans for Proxmox workloads, only the Backup module is available.

You can also back up the Proxmox host separately, as a Linux server. To avoid double backups, you can configure file filers in the protection plan for the host and exclude the directories that contain the virtual machine data.
