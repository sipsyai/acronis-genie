# Selecting Gmail mailboxes

Managing the backup and recovery of workloads and files > Protecting Google Workspace data > Protecting Gmail data > Selecting Gmail mailboxes
Selecting Gmail mailboxes

Select the mailboxes as described below, and then specify other settings of the protection plan as appropriate.

To select Gmail mailboxes

Click Google Workspace.
If multiple Google Workspace organizations were added to the Cyber Protection service, select the organization whose users' data you want to back up. Otherwise, skip this step.

Do one of the following:

To back up the mailboxes of all users (including mailboxes that will be created in the future), expand the Users node, select All users, and then click Group backup.
To back up individual user mailboxes, expand the Users node, select All users, select the users whose mailboxes you want to back up, and then click Backup.

In the backup plan, configure the following:

[If available] In What to back up, select Gmail.

To back up calendars that are shared with the selected users, enable the Include shared calendars switch.

In How long to keep, configure the retention rules for the backups.
In Encryption, click Specify password.
Specify and confirm the encryption password, and then select the encryption algorithm.
Click OK.
