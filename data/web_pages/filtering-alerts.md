To filter the alerts

In the Cyber Protect console, go to Monitoring > Alerts.

In the View drop-down list, select the filtering criteria.

[If you selected Alert category] Select a specific alert category.

Select a specific alert type.

As a result, the alerts that match the filter criteria are listed.
