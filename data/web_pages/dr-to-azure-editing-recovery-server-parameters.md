# Editing the recovery server settings

Implementing disaster recovery > Disaster Recovery to Microsoft Azure > Recovery servers in Microsoft Azure > Editing the recovery server settings
Editing the recovery server settings

When you create and apply a disaster recovery protection plan, a recovery server is created with default settings. You can edit these default settings when necessary.

To edit the settings of a recovery server in Microsoft Azure

In the Cyber Protect console, go to Disaster Recovery > Recovery servers.

Click the server whose settings you want to edit, and then click Edit.

Edit the recovery server settings.

Click Save.
