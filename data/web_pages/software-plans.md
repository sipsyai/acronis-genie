# Software deployment plans

RMM > Managing software > Software deployment plans
Software deployment plans

The availability of this feature depends on the license that you have.

With software deployment plans, you can automate the software deployment process and ensure that the software distribution across your managed workloads is consistent. The functionality includes the following features:

Remote installation or uninstallation of software.
Schedules for automatic installation or uninstallation of software.
Post-installation actions.

Creating a software deployment plan

Adding a workload to a software deployment plan

Removing workloads from software deployment plans

Additional actions with software deployment plans
