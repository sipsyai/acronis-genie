# Adding workloads to a static group

Managing workloads in the Cyber Protect console > Device groups > Adding workloads to a static group
Adding workloads to a static group

You can select the target group first, and then add workloads to it.

Alternatively, you can select the workloads first, and then add them to a group.

To add workloads to a static group

Selecting the target group first
Selecting the workloads first
Click Devices, and then navigate to your target group.
Select the target group, and then click Add devices.
In the folder tree, select the group that contains the required workloads.
Select the check boxes next to the workloads that you want to add, and then click Add.
