# Editing tenant baselines

RMM > Managing and monitoring the Microsoft 365 environment > Working with baselines > Editing tenant baselines
Editing tenant baselines

You can edit baselines that are not managed by a template to change the baseline configuration on the tenant where the baseline is applied.

To edit a baseline

Required role: Partner administrator.

In the Cyber Protect console, go to Microsoft 365 management > Baselines.

Click the pencil icon at the end of the relevant baseline row.

The edit baseline dialog opens.

Apply changes as needed and click Save.

A deviation check starts to assess whether the baseline state is different from the desired state.

Saving the baseline does not change the value in Microsoft 365. It simply saves a configuration target for the baseline.
