Setting the order of favorite plans

You can set the order in which plans that are set as favorite will be shown in the plan selection fields.

You can set up to 10 favorite plans per supported plan type (protection plan, monitoring plan, or remote management plan) per tenant.

The feature is not available for users who are assigned the Read-only administrator role for the Protection service.

Protection plans
Remote management plans
Monitoring plans

Prerequisites

At least two plans are set as favorite.

To set the order of the favorite plans

In the Cyber Protect console plans screen, go to Management > Protection plans.
Click Manage favorites.

Drag and drop the plans to set the order in which they will appear in the plan selection fields.

To be able to drag a plan, click the drag area before the plan's name.

Click Save.
