Enabling or disabling chat notifications in Cyber Protect Monitor

You can enable chat notifications and see a desktop notification when you receive new chat messages. Clicking the notification will open the chat window directly.

Alternatively, if you do not want to see desktop notifications about new chat messages, you can disable chat notifications.

Enable notifications
Disable notifications

To enable chat notifications

Open Cyber Protect Monitor.
Click Settings.
Click Chat notifications.
Select Desktop notifications, and then click Save.
