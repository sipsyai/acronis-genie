# Forwarding archived emails

Managing the backup and recovery of workloads and files > Email archiving > Recovering data from an email archive > Forwarding archived emails
Forwarding archived emails

You can select up to 10 emails from an email archive and send them to recipients of your choice.

To forward archived emails

Log in to the Cyber Protect console as administrator.

Search the email archive or use a saved search query.

For more information, see Searching for emails and Creating a saved search query .

Select up to 10 emails, and then click Send as email.
[If multiple organizations are registered] Select an organization.

Click the Sender field, and then select the user on behalf of which the emails will be sent.

The name of this user appears in the From field of the emails.

In Recipients, type one or more email addresses to which you want to send the selected emails.

If you specify multiple email addresses, separate them by using a comma or semicolon.

Add a description that will be added at the top of all sent emails.

Click Send.

As a result, the selected emails will be sent as individual emails to the email addresses that you specified.
