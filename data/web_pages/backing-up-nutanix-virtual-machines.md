# Backing up Nutanix virtual machines

Installing and deploying Cyber Protection agents > Deploying virtual appliances > Deploying Agent for Nutanix AHV > Backing up Nutanix virtual machines
Backing up Nutanix virtual machines

Prerequisites

You have deployed and configured the Nutanix virtual appliance.

To protect Nutanix virtual machines

Log in to the Cyber Protect console.
Go to Devices > Nutanix > <your cluster>.
Select virtual machines and apply a protection plan to them.
