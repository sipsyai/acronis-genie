Managing tenant baseline templates

Baseline templates enable you to create a reusable set of baselines. They are especially useful when working with multiple tenants, as a template can be managed centrally and allocated to the relevant tenants, as and when required.

Built-in and custom templates

By default, the service provides a couple of built-in templates. You cannot edit or delete these built-in templates, but can use them to create your own custom templates and manage these custom templates as needed. A custom template can contain any mix of baselines.

You need to create a custom template if you require multi-tenant management of baselines.
The default template

By default, the built-in template Best practices is set as default baseline template. The default template is used when onboarding new tenants. A Partner administrator can change the default template as needed.

With Microsoft 365 management, you can manage the baseline templates of your Microsoft 365 organization as follows:
