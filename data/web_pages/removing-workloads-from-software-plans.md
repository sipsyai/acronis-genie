# Removing workloads from software deployment plans

RMM > Managing software > Software deployment plans > Removing workloads from software deployment plans
Removing workloads from software deployment plans

The availability of this feature depends on the license that you have.

You can remove workloads that were added to a software deployment plan.

Prerequisites

2FA is enabled for your user account.
You have one of the following roles for Cyber Protection: Company administrator or Cyber administrator.

To remove workloads from a software deployment plan

In the Cyber Protect console, go to Management > Software deployment plans.
Click the software deployment plan.
Click Manage workloads.
Select one or several workloads that you want to remove from the software deployment plan, and then click Remove.
Click Done.
Click Save.
