# Deleting a recovery server

Implementing disaster recovery > Disaster Recovery to Microsoft Azure > Recovery servers in Microsoft Azure > Deleting a recovery server
Deleting a recovery server

You can delete the recovery servers that you created in Microsoft Azure.

To delete a recovery server in Microsoft Azure

In the Cyber Protect console, go to Disaster Recovery > Recovery servers.

Click the server that you want to delete, and then click Delete.

In the confirmation window, click Delete.
