Applying an archiving plan

You can apply an archiving plan to a mail server.

This operation is only available if you have not yet applied an archiving plan to the mail server. You can apply only one archiving plan to a mail server.

To apply an archiving plan

Log in to the Cyber Protect console as administrator.
Go to Devices > Mail servers.
On the main screen, click the name of the mail server, and then click Protect.
Select an archiving plan, and then click Apply.

As a result, the email archiving plan is applied to the mail server, and the email archive is created.

The initial import will take some time, depending on the selected import scope and period. All new emails will be continuously added to the archive, as they are sent or received.
