Additional actions with chats in Cyber Protect Monitor

If you have an active chat session in Cyber Protect Monitor, you can perform the following actions: edit message, delete message, search for a keyword or message, or end the chat.

Edit message
Delete message
Search in chat
End chat

You can edit any message that you sent in the chat session.

To edit a message in the chat session

In Monitor, in the chat window, locate the message that you want to edit.

Hover over the message, and click the pencil icon that appears.

Edit the message, and then click Save.

The changes are saved. The message is tagged as Edited.
