# Managing and monitoring the Microsoft 365 environment

RMM > Managing and monitoring the Microsoft 365 environment
Managing and monitoring the Microsoft 365 environment

Under service-based licensing, this functionality is part of the paid offering RMM - Microsoft 365 seats. The functionality includes continuous auditing of the Microsoft 365 security posture with best practice baselines, remediation of baseline deviations and user risks, and user onboarding and offboarding. If you do not want to use the Microsoft 365 security monitoring and management service, contact your service provider to disable the integration for you.

Under solution-based licensing, the complete functionality is included in the Security and RMM and the Ultimate Protection license, under the Microsoft 365 seats offering.

A charge is applied for each user allocated with an Exchange Online mailbox license.

The Microsoft 365 security monitoring and management service in Cyber Protection enables you to:

Easily onboard Microsoft 365 customer tenants to Cyber Protect Cloud.
Provide a continuous audit of customers' Microsoft 365 security posture.
Monitor breaches and risks with best practice Microsoft 365 baseline templates.
Remediate baseline deviations (in one-click) and security control misconfigurations.
Monitor and manage Microsoft 365 user risks.
Onboard and offboard Microsoft 365 users.
